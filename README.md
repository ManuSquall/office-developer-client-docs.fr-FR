## <a name="microsoft-open-source-code-of-conduct"></a>Code de conduite Microsoft Open Source
Ce projet a adopté le [code de conduite Microsoft Open Source](https://opensource.microsoft.com/codeofconduct/). Pour plus d’informations, reportez-vous à la [FAQ relative au code de conduite](https://opensource.microsoft.com/codeofconduct/faq/) ou contactez [opencode@microsoft.com](mailto:opencode@microsoft.com) pour toute question ou tout commentaire.

Les documents en direct pour ce repo sont accessibles sur https://docs.microsoft.com/office/client-developer/office-client-development .

Pour plus d’informations sur la façon de contribuer à cette documentation, voir [Contributing](contributing.md).
