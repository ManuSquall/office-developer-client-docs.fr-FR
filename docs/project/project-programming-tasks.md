# <a name="project-programming-tasks"></a>Tâches de programmation du projet

Cette section contient des articles « procédures » qui montrent comment utiliser la bibliothèque JavaScript pour le modèle objet côté client (CSOM) et effectuer d’autres tâches de programmation pour Project Server 2013 et Project Online. Tâches de programmation des exemples de création d’une application hébergée par SharePoint Project Server, création de flux de travail pour la gestion de la demande ; programmation d’applications Project Server avec Windows Communication Foundation (WCF) ; personnalisation du ruban Project Web App ; Création de composants WebPart de Project Server ; Création de gestionnaires d’événements Project Server et les récepteurs d’événements distants ; et en bloc des champs personnalisés de mise à jour et la création de sites de projet pour Project Online.
  
> [!NOTE]
> Pour plus d’informations sur la programmation avec le contrôle de grille JS, voir [JS Grid Control](http://msdn.microsoft.com/en-us/library/ee535898%28office.14%29.aspx) dans la référence du développeur SharePoint 2010. Pour la référence de code managé, voir [Microsoft.SharePoint.JSGrid Namespace](http://msdn.microsoft.com/en-us/library/microsoft.sharepoint.jsgrid%28Office.15%29.aspx). Pour les contrôles web de contrôle de grille JS, voir la [classe JSGrid](http://msdn.microsoft.com/en-us/library/microsoft.sharepoint.webcontrols.jsgrid%28Office.15%29.aspx).
