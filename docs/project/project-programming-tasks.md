# <a name="project-programming-tasks"></a>Tâches de programmation Project

Cette section contient des Articles de procédure qui montrent comment utiliser la bibliothèque JavaScript pour le modèle objet côté client (CSOM) et effectuer d'autres tâches de programmation pour Project Server 2013 et Project online. Des exemples de tâches de programmation incluent la création d'une application Project Server hébergée par SharePoint, la création de flux de travail pour la gestion de la demande; programmation d'applications Project Server avec Windows Communication Foundation (WCF); personnalisation du ruban Project Web App; création de composants WebPart Project Server; la création de gestionnaires d'événements Project Server et de récepteurs d'événements distants; et la mise à jour en bloc des champs personnalisés et la création de sites de projet pour Project online.
  
> [!NOTE]
> Pour plus d'informations sur la programmation avec le contrôle de Grille JS, voir [js Grid Control](https://msdn.microsoft.com/library/ee535898%28office.14%29.aspx) dans la référence du développeur SharePoint 2010. Pour la référence du code managé, voir l' [espace de noms Microsoft. SharePoint. JsGrid](https://msdn.microsoft.com/library/microsoft.sharepoint.jsgrid%28Office.15%29.aspx). Pour les contrôles Web de contrôle de Grille JS, voir [JsGrid Class](https://msdn.microsoft.com/library/microsoft.sharepoint.webcontrols.jsgrid%28Office.15%29.aspx).
