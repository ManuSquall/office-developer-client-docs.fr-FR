# <a name="messagebox-macro-action-access-custom-web-app"></a>Action de Macro contrôle zonemessage. (accès personnalisé web app)

Vous pouvez utiliser l’action de **contrôle zonemessage** pour afficher une boîte de message contenant un avertissement ou un message d’information. Par exemple, vous pouvez utiliser l’action de **contrôle zonemessage** avec des macros de validation. Lorsqu’un contrôle ou un enregistrement échoue une condition de validation de la macro, une boîte de message peut afficher un message d’erreur et fournissent des instructions sur le type de données qui doivent être saisies. 
  
> [!IMPORTANT]
> [!IMPORTANTE] Microsoft ne recommande plus la création et l'utilisation d'applications web Access dans SharePoint. En guise d'alternative, vous pouvez utiliser [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) pour générer des solutions d'entreprise sans code pour le web et les appareils mobiles. 
  
## <a name="setting"></a>Valeur

L’action de **contrôle zonemessage** possède l’argument suivant. 
  
|**Argument de l'action**|**Description**|
|:-----|:-----|
|Message  <br/> |Le texte dans la boîte de message. Vous pouvez taper jusqu'à 255 caractères ou entrer une expression (précédée d’un signe égal).  <br/> |
