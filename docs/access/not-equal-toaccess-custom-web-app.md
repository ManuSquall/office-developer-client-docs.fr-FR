# <a name="not-equal-to-access-custom-web-app"></a>Différent de (application web personnalisée Access)

Compare deux expressions. Lorsque vous comparez des expressions non nulles, le résultat est TRUE si l’opérande gauche a une valeur différente de celle de l’opérande droit ; dans le cas contraire, le résultat est FALSE.
  
> [!IMPORTANT]
> Microsoft ne recommande plus la création et l'utilisation d'applications web Access dans SharePoint. En guise d'alternative, vous pouvez utiliser [Microsoft PowerApps](https://powerapps.microsoft.com/fr-FR/) pour générer des solutions d'entreprise sans code pour le web et les appareils mobiles. 
  
## <a name="syntax"></a>Syntaxe

`< > (Not Equal To)`

*expression*  \<\>  *expression* 
  
*expression*  représente toute expression valide. Les deux expressions doivent avoir des types de données implicitement convertibles. La conversion varie en fonction des règles de priorité des types de données. 
  
## <a name="return-type"></a>Type renvoyé

**Booléen**
