# <a name="-divide-access-custom-web-app"></a>/ (Division) (accès personnalisé web app)

Divise un nombre par un autre.
  
> [!IMPORTANT]
> [!IMPORTANTE] Microsoft ne recommande plus la création et l'utilisation d'applications web Access dans SharePoint. En guise d'alternative, vous pouvez utiliser [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) pour générer des solutions d'entreprise sans code pour le web et les appareils mobiles. 
  
## <a name="syntax"></a>Syntaxe

 *dividende*  /  *diviseur* 
  
 *dividende*  Est l’expression numérique à diviser. Peut être toute expression valide de l’un des types de données de la catégorie de type de données numériques, à l’exception du type de données datetime. 
  
 *Diviseur*  Est l’expression numérique par lequel diviser le dividende. Peut être toute expression valide de l’un des types de données de la catégorie de type de données numériques, à l’exception du type de données datetime. 
  
## <a name="return-type"></a>Type renvoyé

Renvoie le type de données de l’argument avec la priorité la plus élevée. 
  
Si un nombre entier *dividende* est divisé par un entier *diviseur* , le résultat est un entier qui partie décimale du résultat tronqué. 
  
## <a name="remarks"></a>Remarques

La valeur réelle renvoyée par l’opérateur / est le quotient de la première expression divisée par la seconde expression.
