# <a name="group-macro-block-access-custom-web-app"></a>Bloc de Macro groupe (accès personnalisé web app)

L’instruction **Group** vous permet de spécifier un bloc d’actions dans une macro que vous pouvez développer ou réduire. 
  
> [!IMPORTANT]
> [!IMPORTANTE] Microsoft ne recommande plus la création et l'utilisation d'applications web Access dans SharePoint. En guise d'alternative, vous pouvez utiliser [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) pour générer des solutions d'entreprise sans code pour le web et les appareils mobiles. 
  
## <a name="setting"></a>Paramètre

L'instruction **Group** utilise les arguments suivants. 
  
|**Argument**|**Obligatoire**|**Description**|
|:-----|:-----|:-----|
|**Description** <br/> |Non  <br/> |Chaîne qui s’affiche comme titre d’un groupe lorsqu’il est réduit.  <br/> |
   
## <a name="remarks"></a>Notes

L’instruction **Group** ne définit pas une zone d’une macro qui peut être exécutée séparément.
