# <a name="setvariable-macro-action-access-custom-web-app"></a>Action de Macro fonctions SetVariable (accès personnalisé web app)

Vous pouvez utiliser l’action de **fonctions SetVariable** pour créer une variable temporaire et lui attribuer une valeur spécifique. La variable peut ensuite être utilisée en tant que condition ou argument dans les actions suivantes, ou vous pouvez utiliser la variable dans une autre macro de l’interface utilisateur. 
  
> [!IMPORTANT]
> [!IMPORTANTE] Microsoft ne recommande plus la création et l'utilisation d'applications web Access dans SharePoint. En guise d'alternative, vous pouvez utiliser [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) pour générer des solutions d'entreprise sans code pour le web et les appareils mobiles. 
  
## <a name="setting"></a>Valeur

L’action de **fonctions SetVariable** possède les arguments suivants. 
  
|**Argument de l'action**|**Description**|
|:-----|:-----|
|**Variable** <br/> |Entrez le nom de la variable temporaire.  <br/> |
|**Valeur =** <br/> |Expression qui permet de définir la valeur de cette variable temporaire. Ne pas faire précéder l’expression avec l’égal (**=** ) se connecter. Vous pouvez cliquer sur le bouton **Générer** de ![formule](media/buildbut_ZA06047218.gif "formule") à utiliser le Générateur d’Expression pour définir cet argument.  <br/> |
