# <a name="left-function-access-custom-web-app"></a>Fonction Left (application Web personnalisée Access)

Renvoie la partie gauche d'une chaîne de caractères avec le nombre spécifié de caractères.
  
> [!IMPORTANT]
> Microsoft ne recommande plus la création et l'utilisation d'applications web Access dans SharePoint. En guise d'alternative, vous pouvez utiliser [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) pour générer des solutions d'entreprise sans code pour le web et les appareils mobiles. 
  
## <a name="syntax"></a>Syntaxe

 À **gauche** (*TextExpression*, *longueur*) 
  
La fonction **Left** contient les arguments suivants. 
  
|**Nom de l’argument**|**Description**|
|:-----|:-----|
| *TextExpression*  <br/> |Expression de texte.  <br/> |
| *Length*  <br/> |Est un entier positif qui spécifie le nombre de caractères de l' *TextExpression* renvoyés.  <br/> |
