# <a name="-string-concatenation-operator-access-custom-web-app"></a>+ (Opérateur de concaténation de chaînes) (application Web personnalisée Access)

Le signe plus (+) est l'opérateur de concaténation de chaînes qui permet la concaténation de chaînes. Toutes les autres manipulations de chaînes sont gérées à l' **** aide de fonctions de chaîne telles que SUBSTRING. 
  
> [!IMPORTANT]
> Microsoft ne recommande plus la création et l'utilisation d'applications web Access dans SharePoint. En guise d'alternative, vous pouvez utiliser [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) pour générer des solutions d'entreprise sans code pour le web et les appareils mobiles. 
  
## <a name="syntax"></a>Syntaxe

 *TextExpression* **+** *TextExpression* 
  
 *TextExpression*  N'importe quelle expression de texte valide.
