# <a name="editrecord-macro-action-access-custom-web-app"></a>Action de Macro EditRecord (accès personnalisé web app)

Place l’enregistrement actif en mode édition.
  
> [!IMPORTANT]
> [!IMPORTANTE] Microsoft ne recommande plus la création et l'utilisation d'applications web Access dans SharePoint. En guise d'alternative, vous pouvez utiliser [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) pour générer des solutions d'entreprise sans code pour le web et les appareils mobiles. 
  
## <a name="syntax"></a>Syntaxe

 **EditRecord**
  
## <a name="remarks"></a>Remarques

Utilisez l’action **SaveRecord** pour enregistrer les modifications apportées à l’enregistrement ou l’action **UndoRecord** pour ignorer les modifications. 
  
L’action **ModifierEnregistrement** revient à sélectionner le bouton **Modifier** dans la barre d’action.
