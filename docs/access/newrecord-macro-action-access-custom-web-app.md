# <a name="newrecord-macro-action-access-custom-web-app"></a>NewRecord, action de macro (application Web personnalisée Access)

Crée un nouvel enregistrement vide dans la vue active.
  
> [!IMPORTANT]
> Microsoft ne recommande plus la création et l'utilisation d'applications web Access dans SharePoint. En guise d'alternative, vous pouvez utiliser [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) pour générer des solutions d'entreprise sans code pour le web et les appareils mobiles. 
  
## <a name="remarks"></a>Remarques

Vous pouvez utiliser l'action **NewRecord** pour ajouter un enregistrement vide à la fin de l'affichage afin de pouvoir entrer de nouvelles données. 
  
L'action **NewRecord** équivaut à l'ajout d'un enregistrement à l'aide du bouton **Ajouter** de la barre d'action.
