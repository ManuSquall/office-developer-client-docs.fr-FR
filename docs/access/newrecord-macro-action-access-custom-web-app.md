# <a name="newrecord-macro-action-access-custom-web-app"></a>Action de Macro NewRecord (accès personnalisé web app)

Crée un nouvel enregistrement vide dans l’affichage actuel.
  
> [!IMPORTANT]
> [!IMPORTANTE] Microsoft ne recommande plus la création et l'utilisation d'applications web Access dans SharePoint. En guise d'alternative, vous pouvez utiliser [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) pour générer des solutions d'entreprise sans code pour le web et les appareils mobiles. 
  
## <a name="remarks"></a>Remarques

Vous pouvez utiliser l’action **NewRecord** pour ajouter un enregistrement vide à la fin de l’affichage afin que vous pouvez entrer de nouvelles données. 
  
L’action **NewRecord** est équivalente à l’utilisation du bouton **Ajouter** dans la barre d’action pour ajouter un enregistrement.
