# <a name="rand-function-access-custom-web-app"></a>RAND, fonction (accès personnalisé web app)

Renvoie un nombre pseudo-aléatoire compris entre 0 et 1.
  
> [!IMPORTANT]
> [!IMPORTANTE] Microsoft ne recommande plus la création et l'utilisation d'applications web Access dans SharePoint. En guise d'alternative, vous pouvez utiliser [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) pour générer des solutions d'entreprise sans code pour le web et les appareils mobiles. 
  
## <a name="syntax"></a>Syntaxe

 **RAND** ([ *Départ* ]) 
  
La fonction **Rand** contient l’argument suivant. 
  
|**Nom de l’argument**|**Description**|
|:-----|:-----|
| *Seed*  <br/> |Entier qui donne la valeur de départ. Si la *valeur de départ* n’est pas spécifié, une valeur de départ est affectée au hasard.  <br/> |
   
## <a name="remarks"></a>Remarques

Les appels de la fonction **Rand** avec la même valeur initiale répétitives retournent les mêmes résultats.
