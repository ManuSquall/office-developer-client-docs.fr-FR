# <a name="undorecord-macro-action-access-custom-web-app"></a>UndoRecord, action de macro (application Web personnalisée Access)

Ignore les modifications apportées à l'enregistrement actif.
  
> [!IMPORTANT]
> Microsoft ne recommande plus la création et l'utilisation d'applications web Access dans SharePoint. En guise d'alternative, vous pouvez utiliser [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) pour générer des solutions d'entreprise sans code pour le web et les appareils mobiles. 
  
## <a name="remarks"></a>Remarques

L'action **UndoRecord** équivaut à utiliser le bouton **Annuler** de la barre d'action ignorer les modifications.
