# <a name="not-access-custom-web-app"></a>NOT (application Web personnalisée Access)

Nie une entrée booléenne.
  
> [!IMPORTANT]
> Microsoft ne recommande plus la création et l'utilisation d'applications web Access dans SharePoint. En guise d'alternative, vous pouvez utiliser [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) pour générer des solutions d'entreprise sans code pour le web et les appareils mobiles. 
  
## <a name="syntax"></a>Syntaxe

[ **Not** ]  *BooleanExpression* 
  
L'opérateur **not** contient les arguments suivants. 
  
|**Nom de l’argument**|**Description**|
|:-----|:-----|
| *BooleanExpression*  <br/> |Expression booléenne valide.  <br/> |
   
## <a name="remarks"></a>Remarques

Le tableau suivant montre les résultats de la comparaison des valeurs TRUE et FALSe à l'aide de l'opérateur **not** . 
  
||**Not**|
|:-----|:-----|
|**A** <br/> |False  <br/> |
|**TRUE** <br/> |True  <br/> |
