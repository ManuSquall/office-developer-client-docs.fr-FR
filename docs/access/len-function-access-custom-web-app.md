# <a name="len-function-access-custom-web-app"></a>Fonction Len (application Web personnalisée Access)

Renvoie le nombre de caractères de l'expression de chaîne spécifiée, à l'exclusion des espaces de fin.
  
> [!IMPORTANT]
> Microsoft ne recommande plus la création et l'utilisation d'applications web Access dans SharePoint. En guise d'alternative, vous pouvez utiliser [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) pour générer des solutions d'entreprise sans code pour le web et les appareils mobiles. 
  
## <a name="syntax"></a>Syntaxe

 **Len** (*TextExpression*) 
  
La fonction **Len** contient l'argument suivant. 
  
|**Nom de l'argument**|**Description**|
|:-----|:-----|
| *TextExpression*  <br/> |Expression de chaîne.  <br/> |
