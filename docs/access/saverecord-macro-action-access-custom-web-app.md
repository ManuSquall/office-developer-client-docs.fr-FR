# <a name="saverecord-macro-action-access-custom-web-app"></a>SaveRecord, action de macro (application Web personnalisée Access)

Enregistre l'enregistrement actif.
  
> [!IMPORTANT]
> Microsoft ne recommande plus la création et l'utilisation d'applications web Access dans SharePoint. En guise d'alternative, vous pouvez utiliser [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) pour générer des solutions d'entreprise sans code pour le web et les appareils mobiles. 
  
## <a name="remarks"></a>Remarques

L'action **SaveRecord** équivaut à utiliser le bouton **Enregistrer** dans la barre d'action pour enregistrer l'enregistrement actif.
