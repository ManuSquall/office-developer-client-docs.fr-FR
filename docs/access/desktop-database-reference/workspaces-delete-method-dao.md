# <a name="workspacesdelete-method-dao"></a>Méthode Workspaces.Delete (DAO)

**S’applique à**: Access 2013, Office 2013

Supprime le formulaire **Workspace** dans la collection **Workspaces**.

## <a name="syntax"></a>Syntaxe

*expression* . Supprimer (***nom***)

*expression* Variable qui représente un objet **espaces de travail** .
