# <a name="databasetransactions-property-dao"></a>Propriété Database.Transactions (DAO)


**S’applique à**: Access 2013, Office 2013

Renvoie une valeur qui indique si un objet prend en charge les transactions. Valeur **Boolean** en lecture seule.

## <a name="syntax"></a>Syntaxe

*expression* . Transactions

*expression* Variable qui représente un objet de **base de données** .
