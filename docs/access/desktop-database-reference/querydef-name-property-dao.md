# <a name="querydefname-property-dao"></a>Propriété QueryDef.Name (DAO)


**S’applique à** : Access 2013, Office 2013

Cette propriété renvoie ou définit le nom de l'objet spécifié. Type **String** en lecture-écriture.

## <a name="syntax"></a>Syntaxe

*expression* . Nom

*expression* Variable qui représente un objet **QueryDef** .

## <a name="remarks"></a>Remarques

La longueur maximale du nom d'un objet **QueryDef** ne peut dépasser 64 caractères.
