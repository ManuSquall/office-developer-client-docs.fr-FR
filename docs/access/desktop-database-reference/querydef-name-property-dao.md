# <a name="querydefname-property-dao"></a>QueryDef.Name Property (DAO)


**S’applique à**: Access 2013 | Office 2013

Renvoie ou définit le nom de l'objet spécifié. Type **String** en lecture-écriture.

## <a name="syntax"></a>Syntaxe

*expression* . Nom

*expression* Variable qui représente un objet **QueryDef** .

## <a name="remarks"></a>Remarques

La longueur maximale du nom d'un objet **QueryDef** ne peut dépasser 64 caractères.
