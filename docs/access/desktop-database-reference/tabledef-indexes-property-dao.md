# <a name="tabledefindexes-property-dao"></a>Propriété TableDef.Indexes (DAO)


**S’applique à**: Access 2013, Office 2013

Renvoie une collection **Indexes** qui contient tous les objets **Index** stockés pour la table spécifiée. En lecture seule.

## <a name="syntax"></a>Syntaxe

*expression* . Index

*expression* Variable qui représente un objet **TableDef** .
