# <a name="databasename-property-dao"></a>Propriété Database.Name (DAO)


**S’applique à**: Access 2013, Office 2013

Renvoie le nom de l'objet spécifié. Type **String** en lecture seule.

## <a name="syntax"></a>Syntaxe

*expression* . Nom

*expression* Variable qui représente un objet de **base de données** .
