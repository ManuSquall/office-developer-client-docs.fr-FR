# <a name="ado-collections"></a>Collections ADO

**S’applique à**: Access 2013, Office 2013

Chaque collection peut contenir son objet correspondant. Une collection [Errors](error-object-ado.md) peut, par exemple, contenir un objet [Error](errors-collection-ado.md). Pour plus d’informations sur les objets, consultez [objets ADO](ado-objects-and-interfaces.md) ou une rubrique spécifique d’objet.
