# <a name="databaseupdatable-property-dao"></a>Database.Updatable Property (DAO)


**S’applique à**: Access 2013, Office 2013

Renvoie une valeur indiquant si vous pouvez modifier un objet DAO. Type **Boolean** en lecture seule.

## <a name="syntax"></a>Syntaxe

*expression* . Mise à jour

*expression* Variable qui représente un objet de **base de données** .
