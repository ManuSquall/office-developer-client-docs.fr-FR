### <a name="fields-collection"></a><span data-ttu-id="c0808-146">Fields, collection</span><span class="sxs-lookup"><span data-stu-id="c0808-146">Fields collection</span></span>

<span data-ttu-id="c0808-p109">Le paramètre *FieldValue* n’est valide que lorsque vous ajoutez un objet **Field** à un objet [Record](record-object-ado.md) et pas à un objet **Recordset**. Also, the following data types are not supported by ADO: **adIDispatch**, **adIUnknown**, and **adIVariant**.
