# <a name="levelname-property-ado-md"></a>LevelName, propriété (ADO MD)


**S’applique à**: Access 2013, Office 2013

Indique le nom du niveau d'un membre.

## <a name="return-values"></a>Valeurs de retour

Retourne une valeur de type **String** et est en lecture seule.

## <a name="remarks"></a>Notes

Utilisez la propriété **LevelName** pour extraire le nom du niveau auquel appartient un membre. Elle correspond à la propriété [Name](name-property-ado-md.md) d'un objet [Level](level-object-ado-md.md).
