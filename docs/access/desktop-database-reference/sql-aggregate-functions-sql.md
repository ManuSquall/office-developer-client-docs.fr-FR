# <a name="sql-aggregate-functions-sql"></a>Fonctions d'agrégation SQL (SQL)


**S’applique à**: Access 2013, Office 2013

Les fonctions d'agrégation SQL permettent d'effectuer diverses opérations statistiques sur des ensembles de valeurs. Vous pouvez les utiliser dans une requête ou dans des expressions d'agrégation de la propriété **SQL** d'un objet **QueryDef** ou lors de la création d'un objet **Recordset** en fonction d'une requête SQL.

**[Avg, fonction](https://msdn.microsoft.com/library/ff822755\(v=office.15\))**

**[Count, fonction](https://msdn.microsoft.com/library/ff844748\(v=office.15\))**

**[First, Last, fonctions](https://msdn.microsoft.com/library/ff197381\(v=office.15\))**

**[Min, Max, fonctions](https://msdn.microsoft.com/library/ff194490\(v=office.15\))**

**[StDev, StDevP, fonctions](https://msdn.microsoft.com/library/ff197043\(v=office.15\))**

**[Sum, fonction](https://msdn.microsoft.com/library/ff844764\(v=office.15\))**

**[Var, VarP, fonctions](https://msdn.microsoft.com/library/ff192105\(v=office.15\))**
