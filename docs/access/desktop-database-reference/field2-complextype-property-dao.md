# <a name="field2complextype-property-dao"></a>Field2.ComplexType Property (DAO)


**S’applique à**: Access 2013, Office 2013

Renvoie un objet **[ComplexType](complextype-object-dao.md)** qui représente un champ à plusieurs valeurs. En lecture seule.

## <a name="version-information"></a>Informations de version

Version ajoutée : Access 2007

## <a name="syntax"></a>Syntaxe

*expression* . Type complexe

*expression* Variable qui représente un objet **Field2** .
