# <a name="databasecontainers-property-dao"></a>Database.Containers Property (DAO)


**S’applique à**: Access 2013 | Office 2013

Renvoie une collection **Containers** qui représente tous les objets **Container** de la base de données spécifiée. Valeur en lecture seule.

## <a name="syntax"></a>Syntaxe

*expression* . Conteneurs

*expression* Variable qui représente un objet de **base de données** .
