# <a name="understanding-recordset-structure"></a>Présentation de la structure d'un jeu d'enregistrements


**S’applique à**: Access 2013 | Office 2013

Chaque objet **Recordset** possède une collection **Fields** constituée d'un ou plusieurs objets **Field**. Un objet **Field** représente généralement une colonne de table. Les rubriques suivantes expliquent comment naviguer dans la collection **Fields** et obtenir des informations concernant chaque champ. Elles présentent ensuite les types d'informations disponibles par l'intermédiaire de l'objet **Field** et la manière de les utiliser.

  - [Fields, collection](the-fields-collection.md)

  - [Field, objet](the-field-object.md)

  - [Utilisation des jeux d'enregistrements](working-with-recordsets.md)
