# <a name="running-the-address-book-sample-application"></a>L’exemple d’application Carnet d’adresses en cours d’exécution

**S’applique à**: Access 2013, Office 2013

Pour exécuter l'application Carnet d'adresses, procédez comme suit.

**Pour exécuter cette application**

1.  Vérifiez que Microsoft SQL Server est en cours d'exécution. Cliquez sur **Démarrer**, pointez sur **Programmes**, **Microsoft SQL Server 7.0**, puis cliquez sur **Service Manager**. La présence d'une flèche verte dans le cercle blanc indique que SQL Server est en cours d'exécution. Dans le cas contraire (présence d'un carré rouge dans le cercle blanc), cliquez sur **Démarrer/Continuer**.

2.  Dans Microsoft Internet Explorer 4.0 ou version ultérieure, tapez l’adresse suivante : **https://***webserver***/RDS/AddressBook/AddrBook.asp** , où *serveur* est le nom du serveur web où les composants serveur RDS sont installés.

3.  Vous pouvez alors essayer divers scénarios dans l'exemple d'application Carnet d'adresses, par exemple, rechercher une personne à partir de son nom de messagerie électronique, recenser toutes les personnes ayant pour titre « Chef de projet », ou modifier des enregistrements existants. Cliquez sur **Rechercher** pour remplir la grille de données avec tous les noms disponibles.
