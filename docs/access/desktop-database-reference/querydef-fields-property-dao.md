# <a name="querydeffields-property-dao"></a>QueryDef.Fields Property (DAO)


**S’applique à**: Access 2013, Office 2013

Renvoie une collection **[Fields](fields-collection-dao.md)** qui représente tous les objets **[Field](field-object-dao.md)** stockés pour l'objet spécifié. En lecture seule.

## <a name="syntax"></a>Syntaxe

*expression* . Champs

*expression* Variable qui représente un objet **QueryDef** .
