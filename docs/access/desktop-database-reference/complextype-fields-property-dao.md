# <a name="complextypefields-property-dao"></a>Propriété ComplexType.Fields (DAO)


**S’applique à**: Access 2013, Office 2013

Renvoie une collection **Fields** qui représente tous les objets **Field** stockés pour l'objet spécifié. En lecture seule.

## <a name="syntax"></a>Syntaxe

*expression* . Champs

*expression* Variable qui représente un objet **ComplexType** .
