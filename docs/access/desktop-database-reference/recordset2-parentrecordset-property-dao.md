# <a name="recordset2parentrecordset-property-dao"></a>Recordset2.ParentRecordset Property (DAO)


**S’applique à**: Access 2013, Office 2013 

Renvoie l'objet **Recordset** parent de l'objet Recordset défini. En lecture seule.

## <a name="version-information"></a>Informations de version

Version ajoutée : Access 2007

## <a name="syntax"></a>Syntaxe

*expression* . ParentRecordset

*expression* Variable qui représente un objet **Recordset2** .

## <a name="remarks"></a>Remarques

La propriété **ParentRecordset** renvoie la valeur **Null** si l'objet Recordset défini ne représente pas un champ à plusieurs valeurs.
