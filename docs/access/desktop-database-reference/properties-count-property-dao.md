# <a name="propertiescount-property-dao"></a>Propriété Properties.Count (DAO)


**S’applique à**: Access 2013, Office 2013

Vous pouvez utiliser la propriété **Count** pour déterminer le nombre d'éléments dans une collection spécifique. Type **Long** en lecture seule.

## <a name="syntax"></a>Syntaxe

*expression* . Nombre

*expression* Variable qui représente un objet **Properties** .
