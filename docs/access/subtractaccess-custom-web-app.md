# <a name="--subtract-access-custom-web-app"></a>-(Soustraction) (accès personnalisé web app)

Soustrait deux nombres. Cet opérateur arithmétique de soustraction peut également soustraire un nombre, en jours, à partir d’une date.
  
> [!IMPORTANT]
> [!IMPORTANTE] Microsoft ne recommande plus la création et l'utilisation d'applications web Access dans SharePoint. En guise d'alternative, vous pouvez utiliser [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) pour générer des solutions d'entreprise sans code pour le web et les appareils mobiles. 
  
## <a name="syntax"></a>Syntaxe

 *expression*  -  *expression* 
  
 *expression*  Toute expression valide de l’un des types de données de catégorie numérique, à l’exception du type de données **bit** . Ne peut pas être utilisé avec les types de données date ou heure.

Renvoie le type de données de l’argument avec la priorité la plus élevée.
