# <a name="raiseerror-macro-action-access-custom-web-app"></a>Action de Macro Déclenchererreur (accès personnalisé web app)

L’action **Déclenchererreur** affiche une fenêtre contenant un message d’erreur spécifié. 
  
> [!IMPORTANT]
> [!IMPORTANTE] Microsoft ne recommande plus la création et l'utilisation d'applications web Access dans SharePoint. En guise d'alternative, vous pouvez utiliser [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) pour générer des solutions d'entreprise sans code pour le web et les appareils mobiles. 
  
> [!NOTE]
> [!REMARQUE] L'action **DéclencherErreur** est disponible uniquement dans les macros de données. 
  
## <a name="setting"></a>Paramètre

L’action **Déclenchererreur** possède l’argument suivant. 
  
|**Argument**|**Description**|
|:-----|:-----|
| _Description de l’erreur_ <br/> |Expression de type Chaîne qui décrit l’erreur.  <br/> |
   
## <a name="remarks"></a>Notes

Lorsque l’action **Déclenchererreur** est appelée, toutes les opérations dans la transaction en cours sont annulées.
