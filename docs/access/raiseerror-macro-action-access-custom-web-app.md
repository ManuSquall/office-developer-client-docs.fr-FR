# <a name="raiseerror-macro-action-access-custom-web-app"></a>Déclenchererreur, action de macro (application Web personnalisée Access)

L'action **déclenchererreur** affiche une fenêtre contextuelle qui contient un message d'erreur spécifié. 
  
> [!IMPORTANT]
> Microsoft ne recommande plus la création et l'utilisation d'applications web Access dans SharePoint. En guise d'alternative, vous pouvez utiliser [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) pour générer des solutions d'entreprise sans code pour le web et les appareils mobiles. 
  
> [!NOTE]
> L’action **DéclencherErreur** est disponible uniquement dans les macros de données. 
  
## <a name="setting"></a>Setting

L'action **déclenchererreur** possède l'argument suivant. 
  
|**Argument**|**Description**|
|:-----|:-----|
| _Description de l’erreur_ <br/> |Expression de type Chaîne qui décrit l’erreur.  <br/> |
   
## <a name="remarks"></a>Remarques

Lorsque l'action **déclenchererreur** est appelée, toutes les opérations de la transaction en cours sont annulées.
