# <a name="-add-access-custom-web-app"></a>+ (Ajouter) (application Web personnalisée Access)

Additionne deux nombres. Cet opérateur arithmétique d'addition peut également ajouter un nombre, en jours, à une date. 
  
> [!IMPORTANT]
> Microsoft ne recommande plus la création et l'utilisation d'applications web Access dans SharePoint. En guise d'alternative, vous pouvez utiliser [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) pour générer des solutions d'entreprise sans code pour le web et les appareils mobiles. 
  
## <a name="syntax"></a>Syntaxe

 *expression*  +  *expression* 
  
 *expression*  Toute expression valide de l'un des types de données dans la catégorie numérique, à l'exception du type de données **bit** . Ne peut pas être utilisé avec des données de type date ou heure. 
  
## <a name="return-value"></a>Valeur renvoyée

Renvoie le type de données de l'argument dont la priorité est la plus élevée.
