# <a name="floor-function-access-custom-web-app"></a>Fonction Floor (application Web personnalisée Access)

Renvoie le plus grand entier inférieur ou égal à l'expression numérique spécifiée.
  
> [!IMPORTANT]
> Microsoft ne recommande plus la création et l'utilisation d'applications web Access dans SharePoint. En guise d'alternative, vous pouvez utiliser [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) pour générer des solutions d'entreprise sans code pour le web et les appareils mobiles. 
  
## <a name="syntax"></a>Syntaxe

 **Plancher** (*NumericExpression*) 
  
La fonction **Floor** contient l'argument suivant. 
  
|**Nom de l’argument**|**Description**|
|:-----|:-----|
| *NumericExpression*  <br/> |Expression numérique.  <br/> |
