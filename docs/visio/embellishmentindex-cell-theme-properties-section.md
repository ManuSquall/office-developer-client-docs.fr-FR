# <a name="embellishmentindex-cell-theme-properties-section"></a>EmbellishmentIndex, cellule (Section Propriétés de thème)

Modifie l’apparence (ornement) de légendes, les conteneurs, les chronologies et les formes d’organigramme.
  
## <a name="remarks"></a>Note

Pour obtenir une référence à la cellule **EmbellishmentIndex** par un nom à partir d’une autre formule, par la valeur de l’attribut **N** d’un élément de **cellule** ou d’un programme à la propriété **CellsU** , utilisez : 
  
|||
|:-----|:-----|
| Nom de la cellule :  <br/> | EmbellishmentIndex  <br/> |
   
Pour obtenir une référence à la cellule **EmbellishmentIndex** par index dans un programme, utilisez la propriété **CellsSRC** avec les arguments suivants : 
  
|||
|:-----|:-----|
| Index de la section :  <br/> |**visSectionObject** <br/> |
| Index de la ligne :  <br/> |**visRowThemeProperties** <br/> |
| Index de la cellule :  <br/> |**visEmbellishmentIndex** <br/> |
