# <a name="additional-effect-properties-section"></a>Additional Effect Properties Section

Affiche les effets supplémentaires appliqués à la forme, y compris les effets de réflexion, d'éclats et de contours adoucis. Cette section inclut également les valeurs d'un effet d'esquisse.
  
## <a name="remarks"></a>Remarques

Vous pouvez définir les effets réflexion, lumière et contours adoucis dans la boîte de dialogue Format de la **forme** (**effets**de**styles** > de formes**Accueil** > ). Un effet d'esquisse peut être appliqué à une forme à l'aide de l'un des thèmes d'esquisse (**effets**de**thème** > de**conception** > ) ou en modifiant des valeurs dans les cellules ShapeSheet correspondantes.
