# <a name="log10-function"></a>Fonction LOG10

Renvoie le logarithme en base 10 d'un nombre. Le nombre doit être positif ou LOG10 renvoie la valeur d'erreur #NUM.
  
## <a name="syntax"></a>Syntaxe

LOG10 (* * *nombre* * *) 
  
### <a name="parameters"></a>Paramètres

|**Nom**|**Requis/Facultatif**|**Type de données**|**Description**|
|:-----|:-----|:-----|:-----|
| _number_ <br/> |Obligatoire  <br/> |**Numérique** <br/> | Nombre dont vous souhaitez trouver le logarithme en base 10.  <br/> |
   
### <a name="return-value"></a>Valeur renvoyée

Numérique
  
## <a name="example"></a>Exemple

LOG10 (42) 
  
Renvoie 1,6232.
