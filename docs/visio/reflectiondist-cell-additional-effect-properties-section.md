# <a name="reflectiondist-cell-additional-effect-properties-section"></a>ReflectionDist, cellule (Section Propriétés de l’effet supplémentaires)

Détermine la distance que réflexion décalée par rapport à une forme, dans points compris entre 0.0 sur 100,0. 
  
## <a name="remarks"></a>Remarques

Pour obtenir une référence à la cellule **ReflectionDist** par un nom à partir d’une autre formule, par la valeur de l’attribut **N** d’un élément de **cellule** ou d’un programme à la propriété **CellsU** , utilisez : 
  
|||
|:-----|:-----|
| Nom de la cellule :  <br/> | ReflectionDist  <br/> |
   
Pour obtenir une référence à la cellule **ReflectionDist** par index dans un programme, utilisez la propriété **CellsSRC** avec les arguments suivants : 
  
|||
|:-----|:-----|
| Index de la section :  <br/> |**visSectionObject** <br/> |
| Index de la ligne :  <br/> |**visRowOtherEffectProperties** <br/> |
| Index de la cellule :  <br/> |**visReflectionDist** <br/> |
