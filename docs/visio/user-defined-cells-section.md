# <a name="user-defined-cells-section"></a>User-defined Cells, section

Zone de travail pour l'entrée de formules dans des cellules spéciales, susceptibles d'être référencées par d'autres cellules et outils complémentaires.
  
## <a name="remarks"></a>Remarques

Comme la section [Scratch](scratch-section.md) , la section **User-defined Cells** utilisable par votre solution de différentes façons. L’avantage de la section **User-defined Cells** est que les cellules peuvent être nommés ; cellules dans la section **montage** ne peut pas être appelées. 
  
Si votre solution a un objectif bien défini, qu'il est conseillé d’utiliser cellules dans la section **User-defined Cells** pour des raisons de clarté, car les cellules utilisateur peuvent être nommés. La section **Scratch** est généralement utilisée pour isoler les calculs répétés. 
  
Pour ajouter cette section, utilisez la boîte de dialogue **Insérer une Section** (avec le bouton droit dans la fenêtre feuille ShapeSheet et cliquez sur **Insérer une Section**).
