# <a name="lineto-row-geometry-section"></a>LineTo, ligne (section Geometry)

Contient *x* - et *y* -coordonnées du sommet de fin d’un segment de trait droit. 
  
Une ligne LineTo contient les cellules suivantes.
  
|**Cell**|**Description**|
|:-----|:-----|
|[X](x-cell-geometry-section.md) <br/> |*X* -coordonnées du sommet de fin d’un segment de trait droit.  <br/> |
|[Y](y-cell-geometry-section.md) <br/> |La valeur *y* -coordonnées du sommet de fin d’un segment de trait droit.  <br/> |
