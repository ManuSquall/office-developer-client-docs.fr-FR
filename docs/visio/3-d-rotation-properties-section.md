# <a name="3-d-rotation-properties-section"></a>3-D Rotation Properties Section

Affiche les valeurs actuelles de l'angle de rotation d'une forme le long des axes x, y et z.
  
> [!NOTE]
> La section Propriétés de la rotation 3D n'inclut pas les cellules qui contrôlent les effets de relief des formes. Pour définir un effet de relief, modifiez les valeurs des cellules dans la section [Propriétés du biseau](bevel-properties-section.md) . 
  
## <a name="remarks"></a>Remarques

Vous pouvez modifier les valeurs de rotation 3D dans la boîte de dialogue **format** de la forme (**effets** > de**styles** > de forme**Accueil** > ou**rotation 3D** ou modification d'une des cellules dans les propriétés de la **rotation 3D). **section.
