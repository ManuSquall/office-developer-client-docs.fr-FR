# <a name="fill-format-section"></a>Fill Format, section

Décrit les attributs de remplissage de la forme et de son ombre portée (motif, couleurs de premier plan et d’arrière-plan, etc.). 
  
## <a name="remarks"></a>Note

Vous pouvez définir les formats de remplissage dans le volet de **Format de la forme** (sous l’onglet **accueil** , dans le groupe **Styles de forme** , cliquez sur **remplissage**, puis cliquez sur **Options de remplissage**), en appliquant un style de remplissage, ou en créant une entrée dans une cellule dans le **Format de remplissage** section.
