# <a name="reflectionblur-cell-additional-effect-properties-section"></a>ReflectionBlur, cellule (Section Propriétés de l’effet supplémentaires)

Détermine la quantité de flou de réflexion sur une forme, en points, entre 0.0 et 100.0.
  
## <a name="remarks"></a>Remarques

Pour obtenir une référence à la cellule **ReflectionBlur** par un nom à partir d’une autre formule, par la valeur de l’attribut **N** d’un élément de **cellule** ou d’un programme à la propriété **CellsU** , utilisez : 
  
|||
|:-----|:-----|
| Nom de la cellule :  <br/> | ReflectionBlur  <br/> |
   
Pour obtenir une référence à la cellule **ReflectionBlur** par index dans un programme, utilisez la propriété **CellsSRC** avec les arguments suivants : 
  
|||
|:-----|:-----|
| Index de la section :  <br/> |**visSectionObject** <br/> |
| Index de la ligne :  <br/> |**visRowOtherEffectProperties** <br/> |
| Index de la cellule :  <br/> |**visReflectionBlur** <br/> |
