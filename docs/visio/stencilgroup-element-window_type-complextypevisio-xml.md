# <a name="stencilgroup-element-windowtype-complextype-visio-xml"></a>Élément StencilGroup (complexType Window_Type) ('Visio XML')

Spécifie le groupe de fenêtres de stencil fusionné dont la fenêtre est membre.
  
## <a name="element-information"></a>Informations sur l’élément

|||
|:-----|:-----|
|**Type d’élément** <br/> |[StencilGroup_Type](stencilgroup_type-complextypevisio-xml.md) <br/> |
|**Namespace** <br/> |https://schemas.microsoft.com/office/visio/2012/main  <br/> |
|**Fichier de schéma** <br/> |VisioSchema15. xsd  <br/> |
|**Parties de document** <br/> |Windows. Xml  <br/> |
   
## <a name="definition"></a>Définition

```XML
< xs:element name="StencilGroup" type="StencilGroup_Type" minOccurs="0" maxOccurs="1" >
</xs:element >
```

## <a name="elements-and-attributes"></a>Éléments et attributs

Si le schéma définit des exigences spécifiques, telles que **Sequence**, **minOccurs**, **maxOccurs**et **Choice**, reportez-vous à la section définition.
