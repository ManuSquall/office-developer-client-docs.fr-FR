# <a name="stencilgroup-element-windowtype-complextype-visio-xml"></a>Élément StencilGroup (Window_Type, complexType) (« Visio XML »)

Spécifie le groupe de fenêtres de gabarit fusionnée dont la fenêtre est un membre.
  
## <a name="element-information"></a>Informations sur l'élément

|||
|:-----|:-----|
|**Type d’élément** <br/> |[StencilGroup_Type](stencilgroup_type-complextypevisio-xml.md) <br/> |
|**Espace de noms** <br/> |https://schemas.microsoft.com/office/visio/2012/main  <br/> |
|**Fichier de schéma** <br/> |VisioSchema15.xsd  <br/> |
|**Parties de document** <br/> |Windows.Xml  <br/> |
   
## <a name="definition"></a>Définition

```XML
< xs:element name="StencilGroup" type="StencilGroup_Type" minOccurs="0" maxOccurs="1" >
</xs:element >
```

## <a name="elements-and-attributes"></a>Éléments et attributs

Si le schéma définit des exigences spécifiques, telles que **sequence**, **minOccurs**, **maxOccurs**et **choice**, voir la section Définition.
