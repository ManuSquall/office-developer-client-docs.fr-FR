# <a name="bevellightingtype-cell-bevel-properties-section"></a>BevelLightingType, cellule (Section Propriétés de biseau)

Détermine le type d’éclairage utilisé par l’effet de biseau.
  
|**Valeur**|**Description**|
|:-----|:-----|
|0  <br/> |Aucun éclairage  <br/> |
|1  <br/> |Trois points  <br/> |
|2  <br/> |Équilibrer  <br/> |
|3  <br/> |Logiciel  <br/> |
|4  <br/> |Nécessité  <br/> |
|5  <br/> |Flux  <br/> |
|6  <br/> |Contraste  <br/> |
|7  <br/> |Matin  <br/> |
|8  <br/> |Aube  <br/> |
|9  <br/> |Crépuscule  <br/> |
|10  <br/> |Froide  <br/> |
|11  <br/> |Gel  <br/> |
|12  <br/> |À deux dimensions  <br/> |
|13  <br/> |Deux points  <br/> |
|14  <br/> |Lumière  <br/> |
|15  <br/> |Pièce lumineuse  <br/> |
   
## <a name="remarks"></a>Remarques

Pour obtenir une référence à la cellule **BevelLightingType** par un nom à partir d’une autre formule, par la valeur de l’attribut **N** d’un élément de **cellule** ou d’un programme à la propriété **CellsU** , utilisez : 
  
|||
|:-----|:-----|
|Nom de la cellule :  <br/> |BevelLightingType  <br/> |
   
Pour obtenir une référence à la cellule **BevelLightingType** par index dans un programme, utilisez la propriété **CellsSRC** avec les arguments suivants : 
  
|||
|:-----|:-----|
|Index de la section :  <br/> |**visSectionObject** <br/> |
|Index de la ligne :  <br/> |**visRowBevelProperties** <br/> |
|Index de la cellule :  <br/> |**visBevelLightingType** <br/> |
