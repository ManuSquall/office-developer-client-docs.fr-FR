# <a name="bevellightingtype-cell-bevel-properties-section"></a>BevelLightingType Cell (Bevel Properties Section)

Détermine le type d'éclairage utilisé par l'effet de relief.
  
|**Valeur**|**Description**|
|:-----|:-----|
|0  <br/> |Aucun éclairage  <br/> |
|0,1  <br/> |Trois points  <br/> |
|n°2  <br/> |Pense  <br/> |
|3  <br/> |Doux  <br/> |
|4  <br/> |Intensif  <br/> |
|disque  <br/> |Inondations  <br/> |
|6.x  <br/> |Contraste  <br/> |
|7j/7  <br/> |Journée  <br/> |
|8bits  <br/> |Aube  <br/> |
|4,9  <br/> |Crépuscule  <br/> |
|10   <br/> |Froide  <br/> |
|11   <br/> |Libérer  <br/> |
|12   <br/> |À deux dimensions  <br/> |
|13   <br/> |Deux points  <br/> |
|14   <br/> |Lumière  <br/> |
|15   <br/> |Salle lumineuse  <br/> |
   
## <a name="remarks"></a>Remarques

Pour obtenir une référence à la cellule **BevelLightingType** par un nom à partir d'une autre formule, par valeur de l'attribut **N** d'un élément de **cellule** ou d'un programme en faisant appel à la propriété **CellsU** , utilisez: 
  
|||
|:-----|:-----|
|Nom de cellule :  <br/> |BevelLightingType  <br/> |
   
Pour obtenir une référence à la cellule **BevelLightingType** à l'aide d'un index à partir d'un programme, utilisez la propriété **CellsSRC** avec les arguments suivants: 
  
|||
|:-----|:-----|
|Index de la section :  <br/> |**Définis** <br/> |
|Index de la ligne :  <br/> |**visRowBevelProperties** <br/> |
|Index de la cellule :  <br/> |**visBevelLightingType** <br/> |
