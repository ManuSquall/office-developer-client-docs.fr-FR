# <a name="rotationxangle-cell-3-d-rotation-properties-section"></a>RotationXAngle, cellule (Section Propriétés de Rotation 3D)

Détermine l’angle de rotation sur l’axe x, en degrés (0,0 - 359.9). 
  
## <a name="remarks"></a>Remarques

Pour obtenir une référence à la cellule **RotationXAngle** par un nom à partir d’une autre formule, par la valeur de l’attribut **N** d’un élément de **cellule** ou d’un programme à la propriété **CellsU** , utilisez : 
  
|||
|:-----|:-----|
|Nom de la cellule :  <br/> |RotationXAngle  <br/> |
   
Pour obtenir une référence à la cellule **RotationXAngle** par index dans un programme, utilisez la propriété **CellsSRC** avec les arguments suivants : 
  
|||
|:-----|:-----|
|Index de la section :  <br/> |**visSectionObject** <br/> |
|Index de la ligne :  <br/> |**visRow3DRotationProperties** <br/> |
|Index de la cellule :  <br/> |**visRotationXAngle** <br/> |
