# <a name="glowsize-cell-additional-effect-properties-section"></a>GlowSize, cellule (Section Propriétés de l’effet supplémentaires)

Détermine la taille de la lumière externe d’une forme en points. 
  
## <a name="remarks"></a>Remarques

Pour obtenir une référence à la cellule **GlowSize** par un nom à partir d’une autre formule, par la valeur de l’attribut **N** d’un élément de **cellule** ou d’un programme à la propriété **CellsU** , utilisez : 
  
|||
|:-----|:-----|
| Nom de la cellule :  <br/> | GlowSize  <br/> |
   
Pour obtenir une référence à la cellule **GlowSize** par index dans un programme, utilisez la propriété **CellsSRC** avec les arguments suivants : 
  
|||
|:-----|:-----|
| Index de la section :  <br/> |**visSectionObject** <br/> |
| Index de la ligne :  <br/> |**visRowOtherEffectProperties** <br/> |
| Index de la cellule :  <br/> |**visGlowSize** <br/> |
