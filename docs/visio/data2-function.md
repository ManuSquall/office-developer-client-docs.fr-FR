# <a name="data2-function"></a>DATA2, fonction

Renvoie le texte de la zone **données 2** dans la boîte de dialogue **Nom de la forme** d’une forme (sous l’onglet **développeur** , cliquez sur **Nom de la forme** ), sous forme de chaîne. 
  
## <a name="syntax"></a>Syntaxe

DATA2 ()
  
## <a name="remarks"></a>Remarques

Si la zone est vide, la fonction renvoie une chaîne vide.
