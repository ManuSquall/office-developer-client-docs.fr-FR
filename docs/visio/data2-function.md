# <a name="data2-function"></a>Fonction DATA2

Renvoie le texte de la zone **données 2** de la boîte de dialogue nom de la **forme** d'une forme (sous l'onglet **développeur** , cliquez sur nom de la **forme** ), sous la forme d'une chaîne. 
  
## <a name="syntax"></a>Syntaxe

DONNÉES2 ()
  
## <a name="remarks"></a>Remarques

Si la zone est vide, la fonction renvoie une chaîne vide.
