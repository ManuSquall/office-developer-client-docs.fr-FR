# <a name="atan-function"></a>ATAN, fonction

Renvoie l’arc tangente d’un nombre, par exemple, l’angle dont la tangente est _nombre_. L’angle obtenu se situe dans la plage -PI/2 < = angle < = PI/2 radians (-90 < = angle < = 90 degrés). 
  
## <a name="syntax"></a>Syntaxe

ATAN (** *numéro* **) 
  
### <a name="parameters"></a>Paramètres

|**Name**|**Obligatoire/Facultatif**|**Type de données**|**Description**|
|:-----|:-----|:-----|:-----|
| _number_ <br/> |Obligatoire  <br/> |**Numérique** <br/> |Tangente de l’angle.  <br/> |
   
## <a name="example"></a>Exemple

ATAN(1) 
  
Renvoie 45 deg
