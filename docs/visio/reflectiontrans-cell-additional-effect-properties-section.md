# <a name="reflectiontrans-cell-additional-effect-properties-section"></a>ReflectionTrans, cellule (Section Propriétés de l’effet supplémentaires)

Détermine la transparence de la réflexion, sous forme de pourcentage compris entre 0 et 100 %. 
  
## <a name="remarks"></a>Remarques

Pour obtenir une référence à la cellule **ReflectionTrans** par un nom à partir d’une autre formule, par la valeur de l’attribut **N** d’un élément de **cellule** ou d’un programme à la propriété **CellsU** , utilisez : 
  
|||
|:-----|:-----|
| Nom de la cellule :  <br/> | ReflectionTrans  <br/> |
   
Pour obtenir une référence à la cellule **ReflectionTrans** par index dans un programme, utilisez la propriété **CellsSRC** avec les arguments suivants : 
  
|||
|:-----|:-----|
| Index de la section :  <br/> |**visSectionObject** <br/> |
| Index de la ligne :  <br/> |**visRowOtherEffectProperties** <br/> |
| Index de la cellule :  <br/> |**visReflectionTrans** <br/> |
