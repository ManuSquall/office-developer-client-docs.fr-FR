# <a name="about-cell-references"></a><span data-ttu-id="5c2fb-105">À propos des références de cellules</span><span class="sxs-lookup"><span data-stu-id="5c2fb-105">About Cell References</span></span>

<span data-ttu-id="5c2fb-p102">Vous pouvez créer des relations de dépendance entre des formules à l'aide de références de cellule ShapeSheet. Les références de cellule vous permettent de calculer la valeur d'une cellule en fonction de celle d'une autre. Si, par exemple, la cellule Width d'une forme contient une formule permettant de calculer la largeur de la forme en se référant à la valeur de la cellule Height, les proportions de la forme sont conservées si un utilisateur redimensionne la forme verticalement.</span><span class="sxs-lookup"><span data-stu-id="5c2fb-p102">You can create interdependencies among formulas by means of ShapeSheet cell references. Cell references give you the power to calculate a value for one cell based on another cell's value. <span data-ttu-id="5c2fb-109">Dans la formule d’une cellule, vous pouvez faire référence à une cellule de la même forme ou d’un autre objet, par exemple un document ou une page, afin que Microsoft Visio calcule la valeur d’une cellule en fonction de celle de l’autre.</span><span class="sxs-lookup"><span data-stu-id="5c2fb-109">In a cell's formula, you can refer to a cell of the same shape or another object, such as a document or page, so that Microsoft Visio calculates a value for one cell based on another cell's value.</span></span>
