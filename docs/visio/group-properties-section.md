# <a name="group-properties-section"></a>Group Properties Section

Contient les cellules d’un groupe qui contrôlent le mode d’ajout des formes dans un groupe, le déplacement des membres d’un groupe et la sélection des groupes. 
  
## <a name="remarks"></a>Remarques

Vous pouvez également définir ces valeurs sous l’onglet **Comportement** dans la boîte de dialogue **Comportement** (cliquez sur **Comportement** dans le groupe **Création de la forme** sous l’onglet [Développeur](run-in-developer-mode-display-the-developer-tab.md)).
