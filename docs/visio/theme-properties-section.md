# <a name="theme-properties-section"></a>Section Propriétés du thème

Affiche les valeurs d’index du thème appliqué au document, y compris la couleur, la police, connecteur et valeurs d’effets. 
  
## <a name="remarks"></a>Remarques

Vous pouvez définir des valeurs de thème sous l’onglet **Création** ou en apportant des modifications dans les cellules dans la section **Propriétés du thème** de la feuille ShapeSheet.
