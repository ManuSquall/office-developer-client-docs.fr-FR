# <a name="paragraph-section"></a>Paragraph, section

Indique les attributs de mise en forme de paragraphe du texte de la forme, tels que les retraits, l'interligne, les puces et l'alignement horizontal des paragraphes.
  
## <a name="remarks"></a>Note

Formats de paragraphe peuvent être définis en créant une entrée dans une cellule de la section **Paragraph** , en appliquant un style de texte ou en entrant les valeurs dans la boîte de dialogue **texte** (sous l’onglet **accueil** , cliquez sur le Lanceur de boîte de dialogue dans le groupe **police** ). 
  
La section **Paragraph** n’inclut pas les cellules des paramètres de l’onglet. Pour définir les tabulations du texte d’une forme, utilisez la boîte de dialogue **texte** .
