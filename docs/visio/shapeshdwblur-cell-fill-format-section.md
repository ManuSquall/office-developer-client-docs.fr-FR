# <a name="shapeshdwblur-cell-fill-format-section"></a>ShapeShdwBlur, cellule (Section Fill Format)

Détermine la taille du flou de l’ombre d’une forme, en points (0,00 à 100,00). 
  
## <a name="remarks"></a>Remarques

Pour obtenir une référence à la cellule **ShapeShdwBlur** par un nom à partir d’une autre formule, par la valeur de l’attribut **N** d’un élément de **cellule** ou d’un programme à la propriété **CellsU** , utilisez : 
  
|||
|:-----|:-----|
| Nom de la cellule :  <br/> | ShapeShdwBlur  <br/> |
   
Pour obtenir une référence à la cellule **ShapeShdwBlur** par index dans un programme, utilisez la propriété **CellsSRC** avec les arguments suivants : 
  
|||
|:-----|:-----|
| Index de la section :  <br/> |**visSectionObject** <br/> |
| Index de la ligne :  <br/> |**visRowFill** <br/> |
| Index de la cellule :  <br/> |**visFillShdwBlur** <br/> |
