# <a name="sketchlineweight-cell-additional-effect-properties-section"></a>SketchLineWeight, cellule (Section Propriétés de l’effet supplémentaires)

Détermine l’épaisseur supplémentaire ajouté à l’épaisseur de trait à la suite d’un effet esquisse, en points, entre 0 et 50. L’épaisseur de trait d’une forme à mesure que la valeur de l’augmentation de la cellule **SketchLineWeight** . 
  
## <a name="remarks"></a>Remarques

Pour obtenir une référence à la cellule **SketchLineWeight** par un nom à partir d’une autre formule, par la valeur de l’attribut **N** d’un élément de **cellule** ou d’un programme à la propriété **CellsU** , utilisez : 
  
|||
|:-----|:-----|
| Nom de la cellule :  <br/> | SketchLineWeight  <br/> |
   
Pour obtenir une référence à la cellule **SketchLineWeight** par index dans un programme, utilisez la propriété **CellsSRC** avec les arguments suivants : 
  
|||
|:-----|:-----|
| Index de la section :  <br/> |**visSectionObject** <br/> |
| Index de la ligne :  <br/> |**visRowOtherEffectProperties** <br/> |
| Index de la cellule :  <br/> |**visSketchLineWeight** <br/> |
