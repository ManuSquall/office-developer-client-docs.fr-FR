# <a name="colorschemeindex-cell-theme-properties-section"></a>ColorSchemeIndex, cellule (Section Propriétés de thème)

Détermine l’index du thème de couleurs de la forme a après, sous forme d’entier.
  
## <a name="remarks"></a>Remarques

Pour obtenir une référence à la cellule **ColorSchemeIndex** par un nom à partir d’une autre formule, par la valeur de l’attribut **N** d’un élément de **cellule** ou d’un programme à la propriété **CellsU** , utilisez : 
  
|||
|:-----|:-----|
| Nom de la cellule :  <br/> | ColorSchemeIndex  <br/> |
   
Pour obtenir une référence à la cellule **ColorSchemeIndex** par index dans un programme, utilisez la propriété **CellsSRC** avec les arguments suivants : 
  
|||
|:-----|:-----|
| Index de la section :  <br/> |**visSectionObject** <br/> |
| Index de la ligne :  <br/> |**visRowThemeProperties** <br/> |
| Index de la cellule :  <br/> |**visColorSchemeIndex** <br/> |
