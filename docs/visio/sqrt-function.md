# <a name="sqrt-function"></a>SQRT, fonction

Renvoie la racine carrée d’un nombre. 
  
## <a name="syntax"></a>Syntaxe

SQRT (** *numéro* **) 
  
### <a name="parameters"></a>Paramètres

|**Name**|**Obligatoire/Facultatif**|**Type de données**|**Description**|
|:-----|:-----|:-----|:-----|
| _number_ <br/> |Obligatoire  <br/> |**Numérique** <br/> |Nombre dont vous souhaitez trouver la racine carrée.  <br/> |
   
### <a name="return-value"></a>Valeur renvoy�e

Numérique
  
## <a name="remarks"></a>Note

Si _nombre_ est négatif, la fonction SQRT renvoie la valeur d’erreur #NUM !. 
  
## <a name="example"></a>Exemple

SQRT(2) 
  
Renvoie 1,4142.
