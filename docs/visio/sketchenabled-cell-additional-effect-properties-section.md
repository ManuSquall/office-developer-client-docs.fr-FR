# <a name="sketchenabled-cell-additional-effect-properties-section"></a>SketchEnabled, cellule (Section Propriétés de l’effet supplémentaires)

Détermine si un effet esquisse est affiché sur la forme ou non, en tant que valeur de type Boolean. 
  
## <a name="remarks"></a>Remarques

Pour obtenir une référence à la cellule **SketchEnabled** par un nom à partir d’une autre formule, par la valeur de l’attribut **N** d’un élément de **cellule** ou d’un programme à la propriété **CellsU** , utilisez : 
  
|||
|:-----|:-----|
| Nom de la cellule :  <br/> | SketchEnabled  <br/> |
   
Pour obtenir une référence à la cellule **SketchEnabled** par index dans un programme, utilisez la propriété **CellsSRC** avec les arguments suivants : 
  
|||
|:-----|:-----|
| Index de la section :  <br/> |**visSectionObject** <br/> |
| Index de la ligne :  <br/> |**visRowOtherEffectProperties** <br/> |
| Index de la cellule :  <br/> |**visSketchEnabled** <br/> |
