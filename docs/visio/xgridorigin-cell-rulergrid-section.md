# <a name="xgridorigin-cell-ruler-amp-grid-section"></a>Cellule XGridOrigin (règle &amp; Section grille)

Définit la coordonnée horizontale de l'origine de la grille.
  
## <a name="remarks"></a>Remarques

Cette cellule correspond à l' **origine de la grille** horizontale d’option dans le **règle &amp; grille** boîte de dialogue (sous l’onglet **affichage** , cliquez sur la flèche **Afficher** . 
  
Pour obtenir une référence à la cellule XGridOrigin par un nom à partir d’une autre formule ou d’un programme à la propriété **CellsU** , utilisez : 
  
|||
|:-----|:-----|
|Nom de la cellule :  <br/> |XGridOrigin  <br/> |
   
Pour obtenir une référence à la cellule XGridOrigin par index dans un programme, utilisez la propriété **CellsSRC** avec les arguments suivants : 
  
|||
|:-----|:-----|
|Index de la section :  <br/> |**visSectionObject** <br/> |
|Index de la ligne :  <br/> |**visRowRulerGrid** <br/> |
|Index de la cellule :  <br/> |**visXGridOrigin** <br/> |
