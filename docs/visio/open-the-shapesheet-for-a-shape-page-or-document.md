# <a name="open-the-shapesheet-for-a-shape-page-or-document"></a>Ouvrez la feuille ShapeSheet d’une forme, une Page ou un Document

La fenêtre feuille ShapeSheet est masquée par défaut dans Microsoft Visio. Pour accéder à la feuille ShapeSheet, vous devez exécuter Visio en mode développeur (afficher l’onglet Développeur).
  
### <a name="to-run-visio-in-developer-mode"></a>Exécuter Visio en mode développeur

1. Cliquez sur l’onglet **fichier** , puis cliquez sur **Options**.
    
2. Dans la boîte de dialogue **Options Visio** , cliquez sur **Avancé**, puis, sous **Général**, sélectionnez **Exécuter en mode développeur**.
    
### <a name="to-view-the-shapesheet-for-a-shape"></a>Pour afficher la feuille ShapeSheet d’une forme

- Avec le bouton droit de la forme de la zone de dessin, cliquez sur **Afficher la feuille ShapeSheet**.
    
- Sélectionnez la forme de la zone de dessin et puis, sous l’onglet **développeur** , dans le groupe **Création de la forme** , cliquez sur **Afficher la feuille ShapeSheet**, puis cliquez sur **la forme**.
    
### <a name="to-view-the-sheet-for-a-page"></a>Pour afficher la feuille d’une page

- Sous l’onglet **développeur** , dans le groupe **Création de la forme** , cliquez sur **Afficher la feuille ShapeSheet**, puis cliquez sur **Page**.
    
### <a name="to-view-the-sheet-for-a-document"></a>Pour afficher la feuille pour un document

- Sous l’onglet **développeur** , dans le groupe **Création de la forme** , cliquez sur **Afficher la feuille ShapeSheet**, puis cliquez sur **le Document**.
