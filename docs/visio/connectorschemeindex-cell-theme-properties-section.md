# <a name="connectorschemeindex-cell-theme-properties-section"></a>ConnectorSchemeIndex, cellule (Section Propriétés de thème)

Détermine le schéma de connecteur d’un thème est appliqué à la forme, comme un entier. 
  
## <a name="remarks"></a>Remarques

Pour obtenir une référence à la cellule **ConnectorSchemeIndex** par un nom à partir d’une autre formule, par la valeur de l’attribut **N** d’un élément de **cellule** ou d’un programme à la propriété **CellsU** , utilisez : 
  
|||
|:-----|:-----|
| Nom de la cellule :  <br/> | ConnectorSchemeIndex  <br/> |
   
Pour obtenir une référence à la cellule **ConnectorSchemeIndex** par index dans un programme, utilisez la propriété **CellsSRC** avec les arguments suivants : 
  
|||
|:-----|:-----|
| Index de la section :  <br/> |**visSectionObject** <br/> |
| Index de la ligne :  <br/> |**visRowThemeProperties** <br/> |
| Index de la cellule :  <br/> |** visConnectorSchemeIndex ** <br/> |
