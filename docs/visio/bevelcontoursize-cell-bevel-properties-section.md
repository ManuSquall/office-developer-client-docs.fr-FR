# <a name="bevelcontoursize-cell-bevel-properties-section"></a>BevelContourSize, cellule (Section Propriétés de biseau)

Détermine la taille du contour du biseau en points. 
  
## <a name="remarks"></a>Remarques

Pour obtenir une référence à la cellule **BevelContourSize** par un nom à partir d’une autre formule, par la valeur de l’attribut **N** d’un élément de **cellule** ou d’un programme à la propriété **CellsU** , utilisez : 
  
|||
|:-----|:-----|
| Nom de la cellule :  <br/> | BevelContourSize  <br/> |
   
Pour obtenir une référence à la cellule **BevelContourSize** par index dans un programme, utilisez la propriété **CellsSRC** avec les arguments suivants : 
  
|||
|:-----|:-----|
| Index de la section :  <br/> |**visSectionObject** <br/> |
| Index de la ligne :  <br/> |**visRowBevelProperties** <br/> |
| Index de la cellule :  <br/> |**visBevelContourSize** <br/> |
