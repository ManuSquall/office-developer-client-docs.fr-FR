# <a name="quickstyleshadowcolor-cell-quick-style-section"></a>QuickStyleShadowColor, cellule (Section Style rapide)

Détermine quel couleur de thème utilise ombre d’une forme, comme un entier compris entre 0 et 7.
  
|||
|:-----|:-----|
|Valeur  <br/> |Description  <br/> |
|0  <br/> |Couleur de l’ombre de forme hérite de la couleur de thème foncé.  <br/> |
|1  <br/> |Couleur de l’ombre de forme hérite de la couleur de thème clair.  <br/> |
|2  <br/> |Couleur de l’ombre de forme hérite de la couleur de thème Accent 1.  <br/> |
|3  <br/> |Couleur de l’ombre de forme hérite de la couleur de thème Accent 2.  <br/> |
|4  <br/> |Couleur de l’ombre de forme hérite de la couleur de thème Accent 3.  <br/> |
|5  <br/> |Couleur de l’ombre de forme hérite de la couleur de thème Accent 4.  <br/> |
|6  <br/> |Couleur de l’ombre de forme hérite de la couleur de thème Accent 5.  <br/> |
|7  <br/> |Couleur de l’ombre de forme hérite de la couleur de thème Accent 6.  <br/> |
   
## <a name="remarks"></a>Note

Pour obtenir une référence à la cellule **QuickStyleShadowColor** par un nom à partir d’une autre formule, par la valeur de l’attribut **N** d’un élément de **cellule** ou d’un programme à la propriété **CellsU** , utilisez : 
  
|||
|:-----|:-----|
| Nom de la cellule :  <br/> | QuickStyleShadowColor  <br/> |
   
Pour obtenir une référence à la cellule **QuickStyleShadowColor** par index dans un programme, utilisez la propriété **CellsSRC** avec les arguments suivants : 
  
|||
|:-----|:-----|
| Index de la section :  <br/> |**visSectionObject** <br/> |
| Index de la ligne :  <br/> |**visRowQuickStyleProperties** <br/> |
| Index de la cellule :  <br/> |**visQuickStyleShadowColor** <br/> |
