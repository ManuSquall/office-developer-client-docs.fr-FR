# <a name="connection-points-section"></a>Connection Points, section

Contient une ligne et des cellules pour chaque point de connexion défini pour une forme.
  
## <a name="remarks"></a>Remarques

Vous pouvez définir des points de connexion dans la section **Points de connexion** dans la fenêtre feuille ShapeSheet ou en cliquant sur l’outil **Point de connexion** sous l’onglet **accueil** . 
  
Si votre forme n’a pas de section **Points de connexion** , vous pouvez insérer une section **Connection Points** dans la fenêtre feuille ShapeSheet. Avec le bouton droit dans la fenêtre feuille ShapeSheet, cliquez sur **Insérer une Section**, puis cliquez sur **les points de connexion**. Pour ajouter des points de connexion à une section **Connection Points** existante, cliquez sur une ligne, cliquez sur **Insérer une ligne** dans le menu contextuel. 
  
Vous pouvez modifier le type de ligne d’un point de connexion dans cette section pour afficher des cellules supplémentaires que vous pouvez utiliser pour les calculs de montage. Pour ce faire, avec le bouton droit de la ligne, puis cliquez sur **Modifier le Type de ligne** dans le menu contextuel.
