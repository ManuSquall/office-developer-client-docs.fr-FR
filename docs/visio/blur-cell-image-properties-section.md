# <a name="blur-cell-image-properties-section"></a>Blur, cellule (section Image Properties)

Estompe ou adoucit une image bitmap. La valeur par défaut est 0 %.
  
## <a name="remarks"></a>Note

Pour obtenir une référence à la cellule Blur par un nom à partir d’une autre formule ou d’un programme à la propriété **CellsU** , utilisez : 
  
|||
|:-----|:-----|
| Nom de la cellule :  <br/> | Blur  <br/> |
   
Pour obtenir une référence à la cellule Blur par index dans un programme, utilisez la propriété **CellsSRC** avec les arguments suivants : 
  
|||
|:-----|:-----|
| Index de la section :  <br/> |**visSectionObject** <br/> |
| Index de la ligne :  <br/> |**visRowImage** <br/> |
| Index de la cellule :  <br/> |**visImageBlur** <br/> |
