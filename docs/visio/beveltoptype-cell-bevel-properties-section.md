# <a name="beveltoptype-cell-bevel-properties-section"></a>BevelTopType, cellule (Section Propriétés de biseau)

Détermine le type de biseau sur le bord supérieur d’une forme. 
  
|**Valeur**|**Description**|
|:-----|:-----|
|0  <br/> |Aucun biseau  <br/> |
|1  <br/> |Biseau de type cercle  <br/> |
|2  <br/> |Biseau de type Incrustation relâchée  <br/> |
|3  <br/> |Biseau de type croix  <br/> |
|4  <br/> |Cool biseau de type inclinaison  <br/> |
|5  <br/> |Biseau de type angle  <br/> |
|6  <br/> |Biseau de type arrondi souple  <br/> |
|7  <br/> |Biseau de type convexe  <br/> |
|8  <br/> |Biseau de type inclinaison  <br/> |
|9  <br/> |Biseau de type sillage  <br/> |
|10  <br/> |Type Riblet  <br/> |
|11  <br/> |Disque dur biseau Edge  <br/> |
|12  <br/> |Biseau de type Art Déco  <br/> |
   
## <a name="remarks"></a>Remarques

Pour obtenir une référence à la cellule **BevelTopType** par un nom à partir d’une autre formule, par la valeur de l’attribut **N** d’un élément de **cellule** ou d’un programme à la propriété **CellsU** , utilisez : 
  
|||
|:-----|:-----|
|Nom de la cellule :  <br/> |BevelTopType  <br/> |
   
Pour obtenir une référence à la cellule **BevelTopType** par index dans un programme, utilisez la propriété **CellsSRC** avec les arguments suivants : 
  
|||
|:-----|:-----|
|Index de la section :  <br/> |**visSectionObject** <br/> |
|Index de la ligne :  <br/> |**visRowBevelProperties** <br/> |
|Index de la cellule :  <br/> |**visBevelTopType** <br/> |
