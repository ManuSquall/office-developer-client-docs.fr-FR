# <a name="bevel-properties-section"></a>Section Propriétés de biseau

Présente les attributs pour des effets de relief appliquées à la forme. 
  
## <a name="remarks"></a>Remarques

Vous pouvez appliquer un biseau de type à une forme dans le volet de **Format de la forme** (**accueil** > **Styles de forme** > **Bevel** > **Options 3D**, ou en apportant des modifications aux cellules de la section **Propriétés de biseau** de la Feuille ShapeSheet.
