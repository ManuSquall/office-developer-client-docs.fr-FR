# <a name="hyperlinks-section"></a>Hyperlinks, section

Contient les cellules utilisées pour créer des liens entre une forme ou une page de dessin et une autre page de dessin, un autre fichier ou un site Web.
  
## <a name="remarks"></a>Note

Pour définir un lien hypertexte, entrez des valeurs dans les cellules de cette section, ou utilisez la boîte de dialogue **liens hypertexte** (cliquez sur le **lien hypertexte** sous l’onglet **Insertion** ). 
  
Vous pouvez ajouter la section Hyperlinks à une forme, un groupe, une page ou un objet externe dans la fenêtre Feuille ShapeSheet.
