# <a name="headercenter-element-headerfootertype-complextype-visio-xml"></a>HeaderCenter, élément (HeaderFooter_Type, complexType) (« Visio XML »)

Contient la chaîne de texte qui apparaît dans la partie centrale de l’en-tête d’un document.
  
## <a name="element-information"></a>Informations sur l'élément

|||
|:-----|:-----|
|**Type d’élément** <br/> |[HeaderCenter_Type](headercenter_type-complextypevisio-xml.md) <br/> |
|**Espace de noms** <br/> |https://schemas.microsoft.com/office/visio/2012/main  <br/> |
|**Fichier de schéma** <br/> |VisioSchema15.xsd  <br/> |
|**Parties de document** <br/> |document.Xml  <br/> |
   
## <a name="definition"></a>Définition

```XML
< xs:element name="HeaderCenter" type="HeaderCenter_Type" minOccurs="0" maxOccurs="1" >
</xs:element >
```
