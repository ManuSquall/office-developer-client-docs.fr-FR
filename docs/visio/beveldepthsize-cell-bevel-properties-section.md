# <a name="beveldepthsize-cell-bevel-properties-section"></a>BevelDepthSize, cellule (Section Propriétés de biseau)

Détermine la taille de la profondeur de biseau en points. 
  
## <a name="remarks"></a>Remarques

Pour obtenir une référence à la cellule **BevelDepthSize** par un nom à partir d’une autre formule, par la valeur de l’attribut **N** d’un élément de **cellule** ou d’un programme à la propriété **CellsU** , utilisez : 
  
|||
|:-----|:-----|
| Nom de la cellule :  <br/> | BevelDepthSize  <br/> |
   
Pour obtenir une référence à la cellule **BevelDepthSize** par index dans un programme, utilisez la propriété **CellsSRC** avec les arguments suivants : 
  
|||
|:-----|:-----|
| Index de la section :  <br/> |**visSectionObject** <br/> |
| Index de la ligne :  <br/> |**visRowBevelProperties** <br/> |
| Index de la cellule :  <br/> |**visBevelDepthSize** <br/> |
