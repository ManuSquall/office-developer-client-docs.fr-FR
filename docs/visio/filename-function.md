# <a name="filename-function"></a>Fonction FILENAME

Renvoie le nom de fichier d’un document en tant que chaîne.
  
## <a name="syntax"></a>Syntaxe

FILENAME ()
  
### <a name="return-value"></a>Valeur renvoyée

Chaîne
