# <a name="sinh-function"></a>SINH, fonction

Renvoie le sinus hyperbolique d’un angle. 
  
## <a name="syntax"></a>Syntaxe

SINH (** *angle* **) 
  
### <a name="parameters"></a>Paramètres

|**Name**|**Obligatoire/Facultatif**|**Type de données**|**Description**|
|:-----|:-----|:-----|:-----|
| _angle_ <br/> |Obligatoire  <br/> |**Numérique** <br/> |Angle de laquelle obtenir le sinus hyperbolique.  <br/> |
   
### <a name="return-value"></a>Valeur renvoy�e

Nombre
  
## <a name="remarks"></a>Remarques

La fonction SINH utilise des radians, sauf si vous spécifiez une autre unité de mesure dans _angle_.
  
## <a name="example"></a>Exemple

SINH(45 deg) 
  
Renvoie 0,8687.
