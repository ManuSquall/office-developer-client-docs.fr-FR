# <a name="shape-layout-section"></a>Shape Layout, section

Contient les cellules qui définissent le placement des formes et les paramètres de positionnement des liens.
  
## <a name="remarks"></a>Remarques

Vous pouvez également définir les valeurs des cellules spécifiques aux formes dans cette section à l’aide de l’onglet **Placement** dans la boîte de dialogue **comportement** (sous l’onglet [développeur](run-in-developer-mode-display-the-developer-tab.md) , dans le groupe **Création de la forme** , cliquez sur **comportement**). Vous pouvez définir les valeurs des cellules spécifiques aux liens dans cette section dans la boîte de dialogue **comportement** à l’aide de l’onglet **connecteur** . (Pour afficher l’onglet **lien** , sélectionnez le lien dans la fenêtre de dessin, puis afficher la boîte de dialogue **comportement** .) 
  
Vous pouvez utiliser des valeurs dans la section **Shape Layout** pour remplacer les valeurs par défaut contenues dans la section **Page Layout** .
