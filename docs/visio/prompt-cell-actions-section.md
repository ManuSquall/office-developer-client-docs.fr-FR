# <a name="prompt-cell-actions-section"></a>Prompt, cellule (section Actions)

Cette cellule n'est plus utilisée depuis la version Visio 2002.
  
## <a name="remarks"></a>Note

Dans les versions précédentes de Visio, cette cellule définissait le message descriptif qui s'affichait dans la barre d'état lorsque l'utilisateur sélectionnait un raccourci de commande.
