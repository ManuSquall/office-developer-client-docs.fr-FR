# <a name="effectschemeindex-cell-theme-properties-section"></a>EffectSchemeIndex, cellule (Section Propriétés de thème)

Détermine le jeu d’effets du thème appliqué à une forme, sous forme d’entier.
  
## <a name="remarks"></a>Remarques

Pour obtenir une référence à la cellule **EffectSchemeIndex** par un nom à partir d’une autre formule, par la valeur de l’attribut **N** d’un élément de **cellule** ou d’un programme à la propriété **CellsU** , utilisez : 
  
|||
|:-----|:-----|
| Nom de la cellule :  <br/> | EffectSchemeIndex  <br/> |
   
Pour obtenir une référence à la cellule **EffectSchemeIndex** par index dans un programme, utilisez la propriété **CellsSRC** avec les arguments suivants : 
  
|||
|:-----|:-----|
| Index de la section :  <br/> |**visSectionObject** <br/> |
| Index de la ligne :  <br/> |**visRowThemeProperties** <br/> |
| Index de la cellule :  <br/> |** visEffectSchemeIndex ** <br/> |
