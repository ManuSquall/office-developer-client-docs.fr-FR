# <a name="quickstylefontcolor-cell-quick-style-section"></a>QuickStyleFontColor, cellule (Section Style rapide)

Détermine la couleur de police à partir de Styles rapides qui hérite de texte d’une forme à partir du thème actif, comme un entier compris entre 0 et 1. 
  
|||
|:-----|:-----|
|Valeur  <br/> |Description  <br/> |
|0  <br/> |Le texte de forme utilise la couleur de police foncée.  <br/> |
|1  <br/> |Le texte de forme utilise la couleur de police clair.  <br/> |
   
## <a name="remarks"></a>Remarques

Pour obtenir une référence à la cellule **QuickStyleFontColor** par un nom à partir d’une autre formule, par la valeur de l’attribut **N** d’un élément de **cellule** ou d’un programme à la propriété **CellsU** , utilisez : 
  
|||
|:-----|:-----|
| Nom de la cellule :  <br/> | QuickStyleFontColor  <br/> |
   
Pour obtenir une référence à la cellule **QuickStyleFontColor** par index dans un programme, utilisez la propriété **CellsSRC** avec les arguments suivants : 
  
|||
|:-----|:-----|
| Index de la section :  <br/> |**visSectionObject** <br/> |
| Index de la ligne :  <br/> |**visRowQuickStyleProperties** <br/> |
| Index de la cellule :  <br/> |**visQuickStyleFontColor** <br/> |
