# <a name="display-formulas-or-values-in-cells"></a>Affichage de formules ou de valeurs dans les cellules

1. Cliquez dans la fenêtre Feuille ShapeSheet pour l'activer.
    
2. Sous l’onglet **Outils feuille ShapeSheet** , dans le groupe **affichage** , cliquez sur **formules** pour afficher les formules ou sur **valeurs** pour afficher les valeurs. 
    
    > [!NOTE]
    > Vous pouvez également avec le bouton droit dans la fenêtre feuille ShapeSheet, puis sur **formules** ou **valeurs** dans le menu contextuel.
