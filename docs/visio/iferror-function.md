# <a name="iferror-function"></a>IFERROR, fonction

Renvoie le résultat évalué d’une expression primaire, si elle ne correspond pas à une erreur. Sinon, renvoie le résultat d’une autre expression évalué.
  
## <a name="version-information"></a>Informations de version

Version ajoutée : Visio 2010 
  
## <a name="syntax"></a>Syntaxe

IFERROR (** *expression primaire* **, ** *autre expression* **) 
  
### <a name="parameters"></a>Paramètres

|**Name**|**Obligatoire/Facultatif**|**Type de données**|**Description**|
|:-----|:-----|:-----|:-----|
| _expression primaire_ <br/> |Obligatoire  <br/> |**Chaîne** <br/> |Première expression à évaluer.  <br/> |
| _autre expression_ <br/> |Obligatoire  <br/> |**Chaîne** <br/> |Autre expression à évaluer si la première expression évalue une erreur.  <br/> |
