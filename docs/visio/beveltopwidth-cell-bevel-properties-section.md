# <a name="beveltopwidth-cell-bevel-properties-section"></a>BevelTopWidth, cellule (Section Propriétés de biseau)

Détermine la largeur de la plaque du haut exprimée en points. 
  
## <a name="remarks"></a>Remarques

Pour obtenir une référence à la cellule **BevelTopWidth** par un nom à partir d’une autre formule, par la valeur de l’attribut **N** d’un élément de **cellule** ou d’un programme à la propriété **CellsU** , utilisez : 
  
|||
|:-----|:-----|
| Nom de la cellule :  <br/> | BevelTopWidth  <br/> |
   
Pour obtenir une référence à la cellule **BevelTopWidth** par index dans un programme, utilisez la propriété **CellsSRC** avec les arguments suivants : 
  
|||
|:-----|:-----|
| Index de la section :  <br/> |**visSectionObject** <br/> |
| Index de la ligne :  <br/> |**visRowBevelProperties** <br/> |
| Index de la cellule :  <br/> |**visBevelTopWidth** <br/> |
