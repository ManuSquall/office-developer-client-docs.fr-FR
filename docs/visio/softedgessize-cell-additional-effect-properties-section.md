# <a name="softedgessize-cell-additional-effect-properties-section"></a>SoftEdgesSize, cellule (Section Propriétés de l’effet supplémentaires)

Détermine la taille d’une bordure arrondie, en points, entre 0,00 à 100,00. Si la cellule **SoftEdgesSize** a une valeur de 0, la forme n’est pas arrondie. 
  
## <a name="remarks"></a>Remarques

Pour obtenir une référence à la cellule **SoftEdgesSize** par un nom à partir d’une autre formule, par la valeur de l’attribut **N** d’un élément de **cellule** ou d’un programme à la propriété **CellsU** , utilisez : 
  
|||
|:-----|:-----|
| Nom de la cellule :  <br/> | SoftEdgesSize  <br/> |
   
Pour obtenir une référence à la cellule **SoftEdgesSize** par index dans un programme, utilisez la propriété **CellsSRC** avec les arguments suivants : 
  
|||
|:-----|:-----|
| Index de la section :  <br/> |**visSectionObject** <br/> |
| Index de la ligne :  <br/> |**visRowOtherEffectProperties** <br/> |
| Index de la cellule :  <br/> |**visSoftEdgesSize** <br/> |
