# <a name="docmd-function"></a>DOCMD, fonction

Exécute la commande identifiée.
  
## <a name="syntax"></a>Syntaxe

 **DOCMD** ( _commandID_)
  
### <a name="parameters"></a>Paramètres

|**Name**|**Obligatoire/Facultatif**|**Type de données**|**Description**|
|:-----|:-----|:-----|:-----|
| _commandID_ <br/> |Obligatoire  <br/> |**Number** <br/> | Commande à exécuter  <br/> |
   
## <a name="remarks"></a>Remarques

Pour obtenir la liste des commandes prises en charge avec la fonction DOCMD, consultez la rubrique « Commandes DoCmd/DOCMD » dans la référence Automation de Microsoft Visio 2013. 
  
## <a name="example"></a>Exemple

 `DOCMD (1312)`
  
Affiche la boîte de dialogue **Données de forme** apparaissent dans l’interface utilisateur.
