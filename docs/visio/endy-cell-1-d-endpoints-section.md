# <a name="endy-cell-1-d-endpoints-section"></a>EndY, cellule (section 1-D Endpoints)

Représente la valeur *y* -coordonnées du point de terminaison de la forme 1D, par rapport à l’origine de son parent. 
  
## <a name="remarks"></a>Remarques

Pour obtenir une référence à la cellule EndY par un nom à partir d’une autre formule ou d’un programme à la propriété **CellsU** , utilisez : 
  
|||
|:-----|:-----|
| Nom de cellule :  <br/> | EndY  <br/> |
   
Pour obtenir une référence à la cellule EndY par index dans un programme, utilisez la propriété **CellsSRC** avec les arguments suivants : 
  
|||
|:-----|:-----|
| Index de la section :  <br/> |**visSectionObject** <br/> |
| Index de la ligne :  <br/> |**visRowXForm1D** <br/> |
| Index de la cellule :  <br/> |**vis1DEndY** <br/> |
