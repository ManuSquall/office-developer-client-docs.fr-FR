# <a name="bevelmaterialtype-cell-bevel-properties-section"></a>BevelMaterialType, cellule (Section Propriétés de biseau)

Détermine le type de de que biseau se compose. 
  
|**Description**|**Valeur**|
|:-----|:-----|
|0  <br/> |Aucun matériel spécial  <br/> |
|1  <br/> |Mat  <br/> |
|2  <br/> |Mat chaud  <br/> |
|3  <br/> |Plastique  <br/> |
|4  <br/> |Métal  <br/> |
|5  <br/> |Contour foncé  <br/> |
|6  <br/> |Bordure arrondie  <br/> |
|7  <br/> |À deux dimensions  <br/> |
|8  <br/> |Filaire  <br/> |
|9  <br/> |Poudre  <br/> |
|10  <br/> |Poudre transparente  <br/> |
|11  <br/> |Supprimer  <br/> |
   
## <a name="remarks"></a>Remarques

Pour obtenir une référence à la cellule **BevelMaterialType** par un nom à partir d’une autre formule, par la valeur de l’attribut **N** d’un élément de **cellule** ou d’un programme à la propriété **CellsU** , utilisez : 
  
|||
|:-----|:-----|
| Nom de la cellule :  <br/> | BevelMaterialType  <br/> |
   
Pour obtenir une référence à la cellule **BevelMaterialType** par index dans un programme, utilisez la propriété **CellsSRC** avec les arguments suivants : 
  
|||
|:-----|:-----|
| Index de la section :  <br/> |**visSectionObject** <br/> |
| Index de la ligne :  <br/> |**visRowBevelProperties** <br/> |
| Index de la cellule :  <br/> |**visBevelMaterialType** <br/> |
