# <a name="usegroupgradient-cell-gradient-properties-section"></a>UseGroupGradient, cellule (Section Propriétés de dégradé)

Détermine si la forme prend un dégradé lorsque la forme est groupée avec les autres formes, en tant que valeur de type Boolean. La valeur de la cellule **UseGroupGradient** affecte uniquement le remplissage de la forme. 
  
## <a name="remarks"></a>Remarques

Pour obtenir une référence à la cellule **UseGroupGradient** par un nom à partir d’une autre formule, par la valeur de l’attribut **N** d’un élément de **cellule** ou d’un programme à la propriété **CellsU** , utilisez : 
  
|||
|:-----|:-----|
| Nom de la cellule :  <br/> | UseGroupGradient  <br/> |
   
Pour obtenir une référence à la cellule **UseGroupGradient** par index dans un programme, utilisez la propriété **CellsSRC** avec les arguments suivants : 
  
|||
|:-----|:-----|
| Index de la section :  <br/> |**visSectionObject** <br/> |
| Index de la ligne :  <br/> |**visRowGradientProperties** <br/> |
| Index de la cellule :  <br/> |** visUseGroupGradient ** <br/> |
