# <a name="controls-section"></a>Controls, section

Contient une ligne et des cellules pour chaque poignée de contrôle définie pour une forme.
  
## <a name="remarks"></a>Remarques

Vous pouvez définir les poignées de contrôle dans la section **contrôles** dans la fenêtre feuille ShapeSheet. Si votre forme n’a pas de section **Controls** , vous pouvez insérer une dans la fenêtre feuille ShapeSheet. Avec le bouton droit dans la fenêtre feuille ShapeSheet et cliquez sur **Insérer une Section**, puis activez la case à cocher de **contrôles** . Pour ajouter des poignées de contrôle à une section **Controls** existante, cliquez sur une ligne, cliquez sur **Insérer une ligne** dans le menu contextuel.
