# <a name="fontschemeindex-cell-theme-properties-section"></a>FontSchemeIndex Cell (Theme Properties Section

Détermine le jeu de polices d’un thème est appliqué à la forme, comme un entier. 
  
## <a name="remarks"></a>Remarques

Pour obtenir une référence à la cellule **FontSchemeIndex** par un nom à partir d’une autre formule, par la valeur de l’attribut **N** d’un élément de **cellule** ou d’un programme à la propriété **CellsU** , utilisez : 
  
|||
|:-----|:-----|
| Nom de la cellule :  <br/> | FontSchemeIndex  <br/> |
   
Pour obtenir une référence à la cellule **FontSchemeIndex** par index dans un programme, utilisez la propriété **CellsSRC** avec les arguments suivants : 
  
|||
|:-----|:-----|
| Index de la section :  <br/> |**visSectionObject** <br/> |
| Index de la ligne :  <br/> |**visRowThemeProperties** <br/> |
| Index de la cellule :  <br/> |**visFontSchemeIndex** <br/> |
