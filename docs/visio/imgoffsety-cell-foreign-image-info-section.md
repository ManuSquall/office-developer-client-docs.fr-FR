# <a name="imgoffsety-cell-foreign-image-info-section"></a>ImgOffsetY, cellule (section Foreign Image Info)

Détermine que la distance de l’objet décalage vertical entre l’origine de la bordure de l’objet. La valeur par défaut est 0. Panoramique de l’objet avec l’outil **rognage** modifie cette valeur. 
  
## <a name="remarks"></a>Remarques

Pour obtenir une référence à la cellule ImgOffsetY par un nom à partir d’une autre formule ou d’un programme à la propriété **CellsU** , utilisez : 
  
|||
|:-----|:-----|
| Nom de la cellule :  <br/> | ImgOffsetY  <br/> |
   
Pour obtenir une référence à la cellule ImgOffsetY par index dans un programme, utilisez la propriété **CellsSRC** avec les arguments suivants : 
  
|||
|:-----|:-----|
| Index de la section :  <br/> |**visSectionObject** <br/> |
| Index de la ligne :  <br/> |**visRowForeign** <br/> |
| Index de la cellule :  <br/> |**visFrgnImgOffsetY** <br/> |
