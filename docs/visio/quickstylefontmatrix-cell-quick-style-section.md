# <a name="quickstylefontmatrix-cell-quick-style-section"></a>QuickStyleFontMatrix, cellule (Section Style rapide)

Détermine le style de la police pour chaque Style rapide, comme un entier compris entre 1 et 6.
  
## <a name="remarks"></a>Remarques

Pour obtenir une référence à la cellule **QuickStyleFontMatrix** par un nom à partir d’une autre formule, par la valeur de l’attribut **N** d’un élément de **cellule** ou d’un programme à la propriété **CellsU** , utilisez : 
  
|||
|:-----|:-----|
| Nom de la cellule :  <br/> | QuickStyleFontMatrix  <br/> |
   
Pour obtenir une référence à la cellule **QuickStyleFontMatrix** par index dans un programme, utilisez la propriété **CellsSRC** avec les arguments suivants : 
  
|||
|:-----|:-----|
| Index de la section :  <br/> |**visSectionObject** <br/> |
| Index de la ligne :  <br/> |**visRowQuickStyleProperties** <br/> |
| Index de la cellule :  <br/> |**visQuickStyleFontMatrix** <br/> |
