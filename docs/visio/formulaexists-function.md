# <a name="formulaexists-function"></a>FORMULAEXISTS, fonction

Indique si la cellule référencée contient une formule. 
  
## <a name="syntax"></a>Syntaxe

FORMULAEXISTS (** *cellref* **) 
  
### <a name="parameters"></a>Paramètres

|**Name**|**Obligatoire/Facultatif**|**Type de données**|**Description**|
|:-----|:-----|:-----|:-----|
| _CellRef_ <br/> |Obligatoire  <br/> |**Chaîne** <br/> |Cellule dans laquelle vous voulez vérifier la présence d’une formule.  <br/> |
   
## <a name="remarks"></a>Remarques

La fonction FORMULAEXISTS renvoie 1 si la cellule contient une formule ; Si ce n’est pas une formule, elle renvoie zéro (0).
