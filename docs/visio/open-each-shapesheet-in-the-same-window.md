# <a name="open-each-shapesheet-in-the-same-window"></a>Ouvrez toutes les feuilles ShapeSheet dans la même fenêtre

1. Cliquez sur l’onglet **fichier** , cliquez sur **Options**, puis cliquez sur **Avancé**.
    
2. Sous **Général**, sélectionnez **Ouvrir toutes les feuilles ShapeSheet dans la même fenêtre**.
    
3. Cliquez sur **OK**.
