# <a name="copy-and-paste-a-formula-from-one-cell-into-another"></a>Copier-coller d'une formule d'une cellule à une autre

1. Sélectionnez la cellule contenant la formule à copier.
    
2. Appuyez sur CTRL + C.
    
    Pour remplacer l’intégralité de la formule, sélectionnez la cellule dont vous souhaitez remplacer la formule.
    
    Pour coller la formule dans une autre formule, double-cliquez sur la cellule, puis cliquez sur un point d’insertion où vous souhaitez coller la formule.
    
3. Appuyez sur Ctrl+V
    
4. Appuyez sur Entrée pour accepter la formule.
