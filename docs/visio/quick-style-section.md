# <a name="quick-style-section"></a>Quick Style Section

Affiche les valeurs de mise en forme de style rapide actuelles appliquées à la forme, y compris la couleur et le style de trait, la couleur et le style de remplissage, la couleur de l'ombre, les effets, la police et le type de style. 
  
## <a name="remarks"></a>Remarques

Vous pouvez créer un style rapide en sélectionnant la forme, puis en sélectionnant un nouveau style rapide (sous l'onglet **Accueil** , dans le groupe **styles de formes** , cliquez sur **plus**, puis sélectionnez un autre style rapide dans la Galerie).
