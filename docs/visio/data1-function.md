# <a name="data1-function"></a>DATA1, fonction

Renvoie le texte de la zone de **données 1** dans la boîte de dialogue **Nom de la forme** d’une forme (sous l’onglet **développeur** , cliquez sur **Nom de la forme** ), sous forme de chaîne. 
  
## <a name="syntax"></a>Syntaxe

DATA1 ()
  
## <a name="remarks"></a>Remarques

Si la zone est vide, la fonction renvoie une chaîne vide.
