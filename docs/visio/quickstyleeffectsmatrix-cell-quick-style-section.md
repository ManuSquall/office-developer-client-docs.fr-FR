# <a name="quickstyleeffectsmatrix-cell-quick-style-section"></a>QuickStyleEffectsMatrix, cellule (Section Style rapide)

Détermine les effets de styles rapides qui hérite de la forme du thème actif, comme un entier compris entre 0-6. 
  
## <a name="remarks"></a>Remarques

Pour obtenir une référence à la cellule **QuickStyleEffectsMatrix** par un nom à partir d’une autre formule, par la valeur de l’attribut **N** d’un élément de **cellule** ou d’un programme à la propriété **CellsU** , utilisez : 
  
|||
|:-----|:-----|
| Nom de la cellule :  <br/> | QuickStyleEffectsMatrix  <br/> |
   
Pour obtenir une référence à la cellule **QuickStyleEffectsMatrix** par index dans un programme, utilisez la propriété **CellsSRC** avec les arguments suivants : 
  
|||
|:-----|:-----|
| Index de la section :  <br/> |**visSectionObject** <br/> |
| Index de la ligne :  <br/> |**visRowQuickStyleProperties** <br/> |
| Index de la cellule :  <br/> |**visQuickStyleEffectsMatrix** <br/> |
