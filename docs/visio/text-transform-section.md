# <a name="text-transform-section"></a>Text Transform, section

Contient les informations de positionnement du bloc de texte d'une forme.
  
## <a name="remarks"></a>Remarques

Pour ajouter cette section, utilisez la boîte de dialogue **Insérer une Section** (avec le bouton droit dans la fenêtre feuille ShapeSheet et cliquez sur **Insérer une Section**). Définition des valeurs dans la section **Text Transform** est comparable à travailler avec le bloc de texte à l’aide de l’outil **Bloc de texte** (le groupe **Outils** sous l’onglet **accueil** ).
