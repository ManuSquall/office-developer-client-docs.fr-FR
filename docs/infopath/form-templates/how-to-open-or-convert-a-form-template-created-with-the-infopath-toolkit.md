# <a name="open-or-convert-a-form-template-created-with-the-infopath-toolkit"></a><span data-ttu-id="5a803-104">Ouvrir ou de convertir un modèle de formulaire créé avec le Kit de ressources InfoPath</span><span class="sxs-lookup"><span data-stu-id="5a803-104">Open or Convert a Form Template Created with the InfoPath Toolkit</span></span>

<span data-ttu-id="5a803-105">Si vous avez créé un modèle de formulaire InfoPath 2003 avec code managé au moyen de l'un des kits de ressources InfoPath 2003 pour Visual Studio, pour assurer la compatibilité avec InfoPath 2003, vous pouvez continuer à travailler sur ce modèle de formulaire et poursuivre son développement en l'ouvrant dans Microsoft InfoPath ou Visual Studio 2012.</span><span class="sxs-lookup"><span data-stu-id="5a803-105">If you created an InfoPath 2003 managed code form template using one of the InfoPath 2003 Toolkits for Visual Studio and want to maintain compatibility with InfoPath 2003, you can continue to work on and further develop your form template project by opening it in Microsoft InfoPath and Visual Studio 2012.</span></span>
  
<span data-ttu-id="5a803-p101">Une autre solution consiste à migrer et mettre à niveau le code dans votre projet InfoPath 2003 pour utiliser le nouveau modèle objet .NET fourni par l'espace de noms [Microsoft.Office.InfoPath](https://msdn.microsoft.com/library/Microsoft.Office.InfoPath.aspx) . Si vous procédez ainsi, tout votre code devra être réécrit pour utiliser les membres de l'espace de noms **Microsoft.Office.InfoPath**, mais tout le code de votre projet précédent sera conservé et entouré par les instructions **#if InfoPathManagedObjectModel** et **#endif** (C#) ou **#If InfoPathManagedObject Model** et **#End If** (Visual Basic) pour référence.</span><span class="sxs-lookup"><span data-stu-id="5a803-p101">Alternatively, you can migrate and upgrade the code in your InfoPath 2003 project to use the new .NET object model provided by the [Microsoft.Office.InfoPath](https://msdn.microsoft.com/library/Microsoft.Office.InfoPath.aspx) namespace. <span data-ttu-id="5a803-108">Les procédures suivantes décrivent comment ouvrir un modèle de formulaire avec code managé créé avec le kit de ressources InfoPath tout en conservant la compatibilité avec InfoPath 2003 ou comment effectuer la migration et la mise à niveau vers le nouveau modèle objet InfoPath.</span><span class="sxs-lookup"><span data-stu-id="5a803-108">The following procedures describe how to open a managed code form template created by using the InfoPath Toolkit and maintain compatibility with InfoPath 2003 or migrate and upgrade to the new InfoPath object model.</span></span>
