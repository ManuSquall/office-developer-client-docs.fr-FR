 <span data-ttu-id="e430d-151">**Déployer des formulaires et leurs sources de données sur le même domaine**</span><span class="sxs-lookup"><span data-stu-id="e430d-151">**Deploy forms and their data sources on the same domain**</span></span>

 <span data-ttu-id="e430d-155">**Éviter d'utiliser des contrôles ActiveX qui ne sont pas marqués comme étant sécurisés pour les scripts**</span><span class="sxs-lookup"><span data-stu-id="e430d-155">**Avoid using ActiveX controls that are not marked as safe for scripting**</span></span>
