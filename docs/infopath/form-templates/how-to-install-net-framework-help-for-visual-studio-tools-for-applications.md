# <a name="install-local-net-framework-help-for-visual-studio-tools-for-applications"></a>Installer l’aide locale de .NET Framework pour Visual Studio Tools for Applications

Lorsque l'ordinateur sur lequel vous exécutez Microsoft InfoPath et Visual Studio 2012 est connecté à Internet, la visionneuse de l'aide Visual Studio 2012 permet d'accéder à la documentation .NET Framework et aux références des langages Visual Basic et Visual C# à partir de sources en ligne. Toutefois, lorsque l'ordinateur sur lequel vous travaillez n'est pas connecté à Internet, pour accéder à ces sources de documentation, vous devez télécharger et installer une version locale de la documentation .NET Framework 2.0 SDK et effectuer les étapes nécessaires à l'intégration de cette documentation dans l'environnement de développement Visual Studio 2012.
  
Pour afficher une rubrique d'aide qui décrit la procédure à suivre, ouvrez le volet d'aide **Contenu**, développez **Visual Studio for Applications**, développez **Préparation à l'environnement de développement intégré**, puis cliquez sur **Procédure : intégrer la documentation .NET Framework SDK dans l'environnement de développement intégré**.
  
> [!IMPORTANT]
> [!IMPORTANTE] Si l'exécution a lieu sous Windows Vista avec un contrôle de compte d'utilisateur, vous devez démarrer InfoPath comme administrateur avant d'effectuer la procédure référencée ci-dessus. Pour cela, accédez à C:\Program Files\Microsoft Office\Office14, cliquez avec le bouton droit sur INFOPATH.EXE, puis cliquez sur **Exécuter en tant qu'administrateur**. Si vous n'exécutez pas InfoPath en tant qu'administrateur, vous ne pourrez pas effectuer cette procédure.
