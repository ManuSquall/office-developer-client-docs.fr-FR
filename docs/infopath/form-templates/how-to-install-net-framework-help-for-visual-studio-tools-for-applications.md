# <a name="install-local-net-framework-help-for-visual-studio-tools-for-applications"></a><span data-ttu-id="07ccc-104">Installer l’aide locale de .NET Framework pour Visual Studio Tools for Applications</span><span class="sxs-lookup"><span data-stu-id="07ccc-104">Install local .NET Framework Help for Visual Studio Tools for Applications</span></span>

<span data-ttu-id="07ccc-p101">Lorsque l'ordinateur sur lequel vous exécutez Microsoft InfoPath et Visual Studio 2012 est connecté à Internet, la visionneuse de l'aide Visual Studio 2012 permet d'accéder à la documentation .NET Framework et aux références des langages Visual Basic et Visual C# à partir de sources en ligne. Toutefois, lorsque l'ordinateur sur lequel vous travaillez n'est pas connecté à Internet, pour accéder à ces sources de documentation, vous devez télécharger et installer une version locale de la documentation .NET Framework 2.0 SDK et effectuer les étapes nécessaires à l'intégration de cette documentation dans l'environnement de développement Visual Studio 2012.</span><span class="sxs-lookup"><span data-stu-id="07ccc-p101">When the computer on which you are running Microsoft InfoPath and Visual Studio 2012 is connected to the Internet, the Visual Studio 2012 help viewer can access .NET Framework documentation and the Visual Basic and Visual C# language references from online sources. However, when the computer you are working with is not connected to the Internet, you cannot access these sources of documentation until you download and install a local copy of .NET Framework 2.0 SDK documentation and perform the steps required to integrate that documentation with the Visual Studio 2012 development environment.</span></span>
  
<span data-ttu-id="07ccc-107">Pour afficher une rubrique d’aide décrivant la procédure pour ce faire, ouvrez le volet de **contenu** d’aide, développez **Visual Studio for Applications**, développez **Mise en route avec l’environnement de développement intégré**, puis cliquez sur **intégrer .NET Documentation du Kit de développement Framework avec l’environnement de développement intégré**.</span><span class="sxs-lookup"><span data-stu-id="07ccc-107">To view a help topic that describes how to do this, open the help **Contents** pane, expand **Visual Studio for Applications**, expand **Getting Started with the Integrated Development Environment**, and then click **Integrate the .NET Framework SDK Documentation with the Integrated Development Environment**.</span></span>
  
> [!IMPORTANT]
> <span data-ttu-id="07ccc-p102">[!IMPORTANTE] Si l'exécution a lieu sous Windows Vista avec un contrôle de compte d'utilisateur, vous devez démarrer InfoPath comme administrateur avant d'effectuer la procédure référencée ci-dessus. Pour cela, accédez à C:\Program Files\Microsoft Office\Office14, cliquez avec le bouton droit sur INFOPATH.EXE, puis cliquez sur **Exécuter en tant qu'administrateur**. Si vous n'exécutez pas InfoPath en tant qu'administrateur, vous ne pourrez pas effectuer cette procédure.</span><span class="sxs-lookup"><span data-stu-id="07ccc-p102">If you are running under Windows Vista with user account control, you must start InfoPath as an administrator before performing the procedure referenced above. To do that, navigate to C:\Program Files\Microsoft Office\Office14, right-click INFOPATH.EXE, and then click **Run as administrator**. If you are not running InfoPath as an administrator, you will be unable to complete the procedure.</span></span>
