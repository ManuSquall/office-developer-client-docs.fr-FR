# <a name="onenote"></a>OneNote

Recherchez des procédures, des articles techniques, du contenu de référence et des billets de blog pour découvrir des solutions de développement et de personnalisation des applications clientes de bureau OneNote.
  
> [!NOTE]
> [!REMARQUE] Les API décrites dans cette documentation sont conçues uniquement pour les solutions clientes de bureau Win32 OneNote dans les scénarios sans connexion. Pour les scénarios avec connexion, utilisez les API du service OneNote recommandées. Pour plus d'informations, consultez le site [dev.onenote.com](http://dev.onenote.com/).
