<span data-ttu-id="82bfd-p105">Lorsqu'un fichier ou une solution Office est chargé, utilisé, fermé ou génère une erreur dans l'une des applications Office 2013 sélectionnées, celle-ci ajoute un enregistrement dans un magasin de données local (une base de données sur le même ordinateur) comprenant les informations sur l'événement.
