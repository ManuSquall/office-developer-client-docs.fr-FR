# <a name="osc-provider-xml-examples"></a>Exemples de fournisseurs XML OSC

Cette section contient des exemples XML qui utilisent les éléments XML **Capabilities**, **Friends**et **ACTIVITYFEED** , conformes au schéma XML du fournisseur Outlook Social Connector (OSC). 
  
- [Fonctionnalités XML-exemple](capabilities-xml-example.md) présente les **fonctionnalités** XML d'un fournisseur OSC, en spécifiant ses fonctionnalités et ses exigences pour OSC. 
    
- [Friends XML montre comment](friends-xml-example.md) afficher les **amis** XML de deux personnes sur le réseau social. 
    
- L' [exemple XML](activity-feed-xml-example.md) d'informations sur les activités montre le code XML **activityFeed** de quatre activités: les deux mises à jour des images de profil, une mise à jour d'État et un billet de blog. Le code XML spécifie également trois modèles d'affichage d'activité pour l'affichage des activités correspondantes.
