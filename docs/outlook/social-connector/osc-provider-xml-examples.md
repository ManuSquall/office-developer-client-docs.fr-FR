# <a name="osc-provider-xml-examples"></a>Exemples de code XML de fournisseur OSC

Cette section contient des exemples de code XML qui utilisent les éléments XML **activityFeed** , **amis**et **fonctionnalités**, conformes au schéma XML du fournisseur Outlook Social Connector (OSC). 
  
- [Fonctionnalités XML](capabilities-xml-example.md) montre le XML des **fonctionnalités** d’un fournisseur OSC, spécification de ses fonctionnalités et la configuration requise pour l’OSC. 
    
- [Exemple de code XML amis](friends-xml-example.md) montre **amis** XML pour deux personnes sur le réseau social. 
    
- [Activité de flux XML](activity-feed-xml-example.md) montre le code XML de quatre activités **activityFeed** : deux mises à jour de l’image, une mise à jour d’état et un billet de blog de profil. Le code XML spécifie également les trois modèles d’affichage d’activité pour afficher les activités correspondantes.
