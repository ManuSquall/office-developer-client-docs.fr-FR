# <a name="isocialsessionloggedonusername"></a>ISocialSession::LoggedOnUserName

Renvoie une chaîne qui représente le nom d’utilisateur qui est utilisé lors de la connexion.
  
```cpp
[propget] HRESULT _stdcall LoggedOnUserName([out, retval] BSTR* result);
```

## <a name="property-value"></a>Valeur de propriété

Une chaîne qui représente le nom d’utilisateur de l’utilisateur connecté.
