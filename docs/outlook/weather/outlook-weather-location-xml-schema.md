# <a name="outlook-weather-location-xml-schema"></a>Schéma XML d’emplacement météo Outlook

Cette section contient la référence du schéma Outlook 2013 pour le schéma XML d’emplacement météo Outlook. Ce schéma permet aux fournisseurs de services météo de répondre à une demande de service web Outlook pour un code d’emplacement. Le code d’emplacement doit correspondre à l’emplacement pour lequel un utilisateur demande des informations météorologiques dans la barre météorologique.
