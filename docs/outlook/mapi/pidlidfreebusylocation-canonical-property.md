# <a name="pidlidfreebusylocation-canonical-property"></a>Propriété canonique PidLidFreeBusyLocation

  
  
**S’applique à** : Outlook 2013 | Outlook 2016 
  
Spécifie un chemin d’accès d’URL à partir de laquelle un client peut extraire des informations disponible/occupé pour le contact comme un fichier iCal, comme spécifié dans [[MS-OXCICAL]](https://msdn.microsoft.com/library/a685a040-5b69-4c84-b084-795113fb4012%28Office.15%29.aspx).
  
|||
|:-----|:-----|
|Propriétés associées :  <br/> |dispidFreeBusyLocation  <br/> |
|Jeu de propriétés :  <br/> |PSETID_Address  <br/> |
|ID de type long (capot) :  <br/> |0x000080D8  <br/> |
|Type de données :  <br/> |PT_UNICODE  <br/> |
|Domaine :  <br/> |Contact  <br/> |
