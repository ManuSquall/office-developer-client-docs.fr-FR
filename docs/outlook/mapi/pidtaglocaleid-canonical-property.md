# <a name="pidtaglocaleid-canonical-property"></a>Propriété canonique PidTagLocaleId

  
  
**S’applique à**: Outlook 
  
Définit le paramètres régionaux de langue pour les messages générés par le système de traduction, tels que des rapports de remise.
  
|||
|:-----|:-----|
|Propriétés associées :  <br/> |PR_LOCALE_ID  <br/> |
|Identificateur :  <br/> |0x66A1  <br/> |
|Type de données :  <br/> |PT_LONG  <br/> |
|Zone :  <br/> |Divers  <br/> |
   
## <a name="remarks"></a>Remarques

Cette propriété est disponible sur une connexion de la boîte aux lettres privée.
