# <a name="pidlidglobalobjectid-canonical-property"></a>Propriété canonique PidLidGlobalObjectId

  
  
**S’applique à**: Outlook 
  
Spécifie l’identificateur unique de l’objet calendar.
  
|||
|:-----|:-----|
|Propriétés associées :  <br/> |LID_GLOBAL_OBJID  <br/> |
|Jeu de propriétés :  <br/> |PSETID_Meeting  <br/> |
|ID de type long (capot) :  <br/> |0 x 00000003  <br/> |
|Type de données :  <br/> |PT_BINARY  <br/> |
|Zone :  <br/> |Réunions  <br/> |
   
## <a name="remarks"></a>Remarques

Une fois définies pour un objet calendar, la valeur de cette propriété ne doit pas changer. Vous trouverez une description détaillée du format dans [[MS-OXOCAL]](http://msdn.microsoft.com/library/9b636532-9150-4836-9635-9c9b756c9ccf%28Office.15%29.aspx).
