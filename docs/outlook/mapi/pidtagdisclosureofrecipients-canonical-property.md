# <a name="pidtagdisclosureofrecipients-canonical-property"></a>Propriété canonique PidTagDisclosureOfRecipients

  
  
**S’applique à**: Outlook 
  
Contient la valeur TRUE si la divulgation de destinataires est autorisée.
