# <a name="pidtagradiotelephonenumber-canonical-property"></a>Propriété canonique PidTagRadioTelephoneNumber

  
  
**S’applique à**: Outlook 
  
Contient le numéro de radiotéléphone du destinataire.
  
|||
|:-----|:-----|
|Propriétés associées :  <br/> |PR_RADIO_TELEPHONE_NUMBER, PR_RADIO_TELEPHONE_NUMBER_A, PR_RADIO_TELEPHONE_NUMBER_W  <br/> |
|Identificateur :  <br/> |0x3A1D  <br/> |
|Type de données :  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Zone :  <br/> |Utilisateur de messagerie MAPI  <br/> |
   
## <a name="remarks"></a>Remarques

Ces propriétés permettent d’identifier et accéder aux informations pour un destinataire. Ils sont définis par le destinataire et l’organisation du destinataire.
