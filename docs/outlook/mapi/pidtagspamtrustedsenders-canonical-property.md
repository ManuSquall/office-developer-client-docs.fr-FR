# <a name="pidtagspamtrustedsenders-canonical-property"></a>Propriété canonique PidTagSpamTrustedSenders

  
  
**S’applique à**: Outlook 
  
Contient une liste délimitée par des points-virgules, des adresses de messagerie et des domaines qui sont composés d’expéditeurs approuvés.
  
|||
|:-----|:-----|
|Propriétés associées :  <br/> |PR_SPAM_TRUSTED_SENDERS_W  <br/> |
|ID de type long (capot) :  <br/> |0x0418  <br/> |
|Type de données :  <br/> |PT_UNICODE  <br/> |
|Zone :  <br/> |Courrier indésirable  <br/> |
