# <a name="pidlidworkaddressstate-canonical-property"></a>Propriété canonique PidLidWorkAddressState

  
  
**S’applique à** : Outlook 2013 | Outlook 2016 
  
Spécifie la partie département ou région de l'adresse postale professionnelle du contact.
  
|||
|:-----|:-----|
|Propriétés associées :  <br/> |dispidWorkAddressState  <br/> |
|Jeu de propriétés:  <br/> |PSETID_Address  <br/> |
|ID long (couvercle):  <br/> |0x00008047  <br/> |
|Type de données :  <br/> |PT_UNICODE  <br/> |
|Domaine :  <br/> |Contact  <br/> |
