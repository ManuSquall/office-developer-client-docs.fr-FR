# <a name="pidlidfax2emailaddress-canonical-property"></a>Propriété canonique PidLidFax2EmailAddress

  
  
**S’applique à** : Outlook 2013 | Outlook 2016 
  
Spécifie l'adresse de messagerie de l'adresse de télécopie personnelle du contact.
  
|||
|:-----|:-----|
|Propriétés associées :  <br/> |dispidFax2EmailAddress  <br/> |
|Jeu de propriétés:  <br/> |PSETID_Address  <br/> |
|ID long (couvercle):  <br/> |0x000080C3  <br/> |
|Type de données :  <br/> |PT_UNICODE  <br/> |
|Domaine :  <br/> |Contact  <br/> |
   
## <a name="remarks"></a>Remarques

Cette propriété, si elle est présente, doit contenir un nom complet lisible par l'utilisateur, suivi du caractère «@», suivi d'un numéro de télécopie.
