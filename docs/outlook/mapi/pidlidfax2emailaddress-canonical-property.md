# <a name="pidlidfax2emailaddress-canonical-property"></a>Propriété canonique PidLidFax2EmailAddress

  
  
**S’applique à**: Outlook 
  
Spécifie l’adresse de messagerie de l’adresse de télécopie (domicile).
  
|||
|:-----|:-----|
|Propriétés associées :  <br/> |dispidFax2EmailAddress  <br/> |
|Jeu de propriétés :  <br/> |PSETID_Address  <br/> |
|ID de type long (capot) :  <br/> |0x000080C3  <br/> |
|Type de données :  <br/> |PT_UNICODE  <br/> |
|Zone :  <br/> |Contact  <br/> |
   
## <a name="remarks"></a>Remarques

Cette propriété, le cas échéant, doit contenir un nom complet explicite de l’utilisateur, suivi par le « @ » caractère, suivi d’un numéro de télécopie.
