# <a name="developing-a-mapi-transport-provider"></a>Développement d’un fournisseur de Transport MAPI

  
  
**S’applique à**: Outlook 
  
Vous devez être familiarisé avec l’architecture du sous-système MAPI, le langage C++ et avec l’écriture des bibliothèques de liens dynamiques (DLL) pour les différentes plateformes Windows avant de développer un fournisseur de transport.
