# <a name="pidtagipmtaskentryid-canonical-property"></a>Propriété canonique PidTagIpmTaskEntryId

  
  
**S’applique à** : Outlook 2013 | Outlook 2016 
  
Contient la propriété **EntryID** du dossier tâches d'Outlook. 
  
|||
|:-----|:-----|
|Propriétés associées :  <br/> |PR_IPM_TASK_ENTRYID  <br/> |
|Identificateur :  <br/> |0x36D4  <br/> |
|Type de données :  <br/> |PT_BINARY  <br/> |
|Domaine :  <br/> |Folder  <br/> |
   
## <a name="remarks"></a>Remarques

Cette propriété est en lecture ou en écriture à l'aide du protocole de propriété et de l'objet Stream. Il est lu et écrit dans la boîte de réception ou dans le dossier racine. L'implémentation doit utiliser le dossier boîte de réception quand le magasin est celui de l'utilisateur de la messagerie principale, et qu'il doit utiliser le dossier racine lorsque le magasin est celui d'un utilisateur délégué.
