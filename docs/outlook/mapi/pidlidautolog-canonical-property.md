# <a name="pidlidautolog-canonical-property"></a>Propriété canonique PidLidAutoLog

  
  
**S’applique à**: Outlook 
  
Spécifie s’il faut créer un journal pour chaque action associée à ce contact.
  
|||
|:-----|:-----|
|Propriétés associées :  <br/> |dispidAutoLog  <br/> |
|Jeu de propriétés :  <br/> |PSETID_Address  <br/> |
|ID de type long (capot) :  <br/> |0x00008025  <br/> |
|Type de données :  <br/> |PT_BOOLEAN  <br/> |
|Zone :  <br/> |Contact  <br/> |
