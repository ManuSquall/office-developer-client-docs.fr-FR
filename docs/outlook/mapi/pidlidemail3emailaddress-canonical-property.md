# <a name="pidlidemail3emailaddress-canonical-property"></a>Propriété canonique PidLidEmail3EmailAddress

  
  
**S’applique à**: Outlook 2013 | Outlook 2016 
  
Spécifie l’adresse de messagerie tiers du contact.
  
|||
|:-----|:-----|
|Propriétés associées :  <br/> |dispidEmail3EmailAddress  <br/> |
|Jeu de propriétés :  <br/> |PSETID_Address  <br/> |
|ID de type long (capot) :  <br/> |0x000080A3  <br/> |
|Type de données :  <br/> |PT_UNICODE  <br/> |
|Domaine :  <br/> |Contact  <br/> |
   
## <a name="remarks"></a>Remarques

La valeur de cette propriété doit être appropriée pour le type d’adresse pour cette adresse de messagerie.
