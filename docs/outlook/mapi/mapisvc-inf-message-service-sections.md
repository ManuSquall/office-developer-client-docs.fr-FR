# <a name="mapisvcinf-message-service-sections"></a>Sections de Service de Message MapiSvc.inf

  
  
**S’applique à**: Outlook 
  
MAPISVC.inf comprend une section de service de message pour chacune des entrées figure dans la section **[Services]** . Il existe deux types d’entrées dans les sections suivantes : 1 pour définir certaines propriétés et l’autre pour la liste des noms des sections qui sont liées au service de message en cours de configuration.
