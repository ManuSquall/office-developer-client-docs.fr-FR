# <a name="mapisvcinf-message-service-sections"></a>Sections du service de messagerie MapiSvc. inf

  
  
**S’applique à** : Outlook 2013 | Outlook 2016 
  
MAPISVC. inf inclut une section service de messages pour chacune des entrées indiquées dans la section **[services]** . Il existe deux types d'entrées dans les sections suivantes: une pour définir certaines propriétés et l'autre pour répertorier les noms des sections liées au service de messagerie en cours de configuration.
