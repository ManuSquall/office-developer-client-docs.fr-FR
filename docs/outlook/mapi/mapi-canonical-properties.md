# <a name="mapi-canonical-properties"></a><span data-ttu-id="45e56-103">Propriétés canoniques MAPI</span><span class="sxs-lookup"><span data-stu-id="45e56-103">MAPI Canonical Properties</span></span>

<span data-ttu-id="45e56-105">Une propriété canonique est une propriété virtuelle qui représente une propriété MAPI ou plusieurs propriétés MAPI définies avec le même identificateur de propriété.</span><span class="sxs-lookup"><span data-stu-id="45e56-105">A canonical property is a virtual property that represents a MAPI property, or multiple MAPI properties defined with the same property identifier.</span></span> <span data-ttu-id="45e56-106">Les propriétés canoniques visent uniquement à faciliter l'identification cohérente des propriétés MAPI dans les discussions ou la documentation en dehors du code.</span><span class="sxs-lookup"><span data-stu-id="45e56-106">Canonical properties are only intended to facilitate consistent identification of MAPI properties in discussions or documentation outside of code.</span></span> <span data-ttu-id="45e56-107">Contrairement aux noms de propriété balisés définis par MAPI, les noms de propriétés canoniques ne sont pas définis comme constantes globales dans les fichiers d'en-tête MAPI.</span><span class="sxs-lookup"><span data-stu-id="45e56-107">Unlike MAPI-defined tagged property names, canonical property names are not defined as global constants in MAPI header files.</span></span>
  
## <a name="naming-conventions"></a><span data-ttu-id="45e56-108">Conventions d'affectation de noms</span><span class="sxs-lookup"><span data-stu-id="45e56-108">Naming Conventions</span></span>

<span data-ttu-id="45e56-109">Les noms de propriété canoniques commencent par un préfixe, «PID», qui représente «identificateur de propriété».</span><span class="sxs-lookup"><span data-stu-id="45e56-109">Canonical property names begin with a prefix, "Pid", which represents "property identifier."</span></span> <span data-ttu-id="45e56-110">Selon qu'il s'agit d'une propriété balisée, d'une propriété nommée avec un identificateur numérique ou d'une propriété nommée avec un nom de chaîne, le préfixe est qualifié par «PidTag», «PidLid» et «PidName» respectivement.</span><span class="sxs-lookup"><span data-stu-id="45e56-110">Depending on whether the property is a tagged property, a named property with a numerical identifier, or a named property with a string name, the prefix is further qualified as "PidTag," "PidLid," and "PidName" respectively.</span></span> <span data-ttu-id="45e56-111">Par exemple, [PidTagAccount](pidtagaccount-canonical-property.md) représente les propriétés marquées, **PR_ACCOUNT** ([PidTagAccount](pidtagaccount-canonical-property.md)) **, PR_ACCOUNT_A** ([PidTagAccount](pidtagaccount-canonical-property.md)) et **PR_ACCOUNT_W** ([PidTagAccount](pidtagaccount-canonical-property.md)), qui spécifient le destinataire nom du compte; [PidLidContacts](pidlidcontacts-canonical-property.md) représente la propriété **dispidContacts** , une propriété nommée qui a un identificateur numérique et spécifie le nom des contacts associés à un message; et [PidNamePhishingStamp](pidnamephishingstamp-canonical-property.md) représente "https://schemas.microsoft.com/outlook/phishingstamp," une propriété nommée qui a un nom de chaîne et spécifie la chaîne marquant les messages susceptibles d'être du hameçonnage.</span><span class="sxs-lookup"><span data-stu-id="45e56-111">For example, [PidTagAccount](pidtagaccount-canonical-property.md) represents the tagged properties, **PR_ACCOUNT** ([PidTagAccount](pidtagaccount-canonical-property.md)), **PR_ACCOUNT_A** ([PidTagAccount](pidtagaccount-canonical-property.md)), and **PR_ACCOUNT_W** ([PidTagAccount](pidtagaccount-canonical-property.md)), that specify a recipient's account name; [PidLidContacts](pidlidcontacts-canonical-property.md) represents the **dispidContacts** property, a named property that has a numerical identifier and that specifies the name of contacts associated with a message; and [PidNamePhishingStamp](pidnamephishingstamp-canonical-property.md) represents "https://schemas.microsoft.com/outlook/phishingstamp," a named property that has a string name, and that specifies the string marking messages that are likely to be phishing.</span></span> 
  
## <a name="representing-similar-properties-using-one-canonical-property"></a><span data-ttu-id="45e56-112">Représentation de propriétés similaires à l'aide d'une propriété canonique</span><span class="sxs-lookup"><span data-stu-id="45e56-112">Representing Similar Properties Using One Canonical Property</span></span>

### <a name="identifying-properties-in-mapi"></a><span data-ttu-id="45e56-113">Identification des propriétés dans MAPI</span><span class="sxs-lookup"><span data-stu-id="45e56-113">Identifying Properties in MAPI</span></span>

<span data-ttu-id="45e56-114">Toutes les propriétés de MAPI sont identifiées par un identificateur de propriété qui est une valeur 16 bits non signée.</span><span class="sxs-lookup"><span data-stu-id="45e56-114">All properties in MAPI are identified by a property identifier that is an unsigned 16-bit value.</span></span> <span data-ttu-id="45e56-115">L'identificateur de propriété et le type de propriété (une autre valeur 16 bits non signée) constituent une balise de propriété pour la propriété.</span><span class="sxs-lookup"><span data-stu-id="45e56-115">The property identifier and the property type (another unsigned 16-bit value) constitute a property tag for the property.</span></span> 
  
<span data-ttu-id="45e56-116">MAPI utilise une balise de propriété pour définir des propriétés de manière unique.</span><span class="sxs-lookup"><span data-stu-id="45e56-116">MAPI uses a property tag to uniquely define properties.</span></span> <span data-ttu-id="45e56-117">Les propriétés qui ont la même balise property, comme **PR_BUSINESS2_TELEPHONE_NUMBER** ([PidTagBusiness2TelephoneNumber](pidtagbusiness2telephonenumber-canonical-property.md)) et **PR_OFFICE2_TELEPHONE_NUMBER** ([PidTagBusiness2TelephoneNumber](pidtagbusiness2telephonenumber-canonical-property.md)), sont considérées comme identiques Propriétés dans MAPI.</span><span class="sxs-lookup"><span data-stu-id="45e56-117">Properties that have the same property tag, like **PR_BUSINESS2_TELEPHONE_NUMBER** ([PidTagBusiness2TelephoneNumber](pidtagbusiness2telephonenumber-canonical-property.md)) and **PR_OFFICE2_TELEPHONE_NUMBER** ([PidTagBusiness2TelephoneNumber](pidtagbusiness2telephonenumber-canonical-property.md)), are considered identical properties in MAPI.</span></span> <span data-ttu-id="45e56-118">Pour obtenir la liste des balises de propriété que MAPI a définies pour ses propres propriétés, voir le fichier d'en-tête MAPI Mapitags. h.</span><span class="sxs-lookup"><span data-stu-id="45e56-118">For a list of property tags that MAPI has defined for its own properties, see the MAPI header file, Mapitags.h.</span></span>
  
<span data-ttu-id="45e56-119">Notez que MAPI divise les identificateurs de propriétés en plages.</span><span class="sxs-lookup"><span data-stu-id="45e56-119">Note that MAPI divides property identifiers into ranges.</span></span> <span data-ttu-id="45e56-120">L'emplacement où se trouve un identificateur dans la plage indique son utilisation et sa propriété.</span><span class="sxs-lookup"><span data-stu-id="45e56-120">Where an identifier falls in the range indicates its use and ownership.</span></span> <span data-ttu-id="45e56-121">Les identificateurs de propriété des propriétés balisées se situent dans la plage de 0x0001 à 0x7FFF.</span><span class="sxs-lookup"><span data-stu-id="45e56-121">The property identifiers of tagged properties fall in the range of 0x0001 to 0x7FFF.</span></span> <span data-ttu-id="45e56-122">Dans cette plage se trouvent les identificateurs de propriété des propriétés définies par MAPI, qui se situent dans la plage 0x0001 à 0x3FFF.</span><span class="sxs-lookup"><span data-stu-id="45e56-122">Within this range are the property identifiers of MAPI-defined properties, which fall in the range of 0x0001 to 0x3FFF.</span></span> <span data-ttu-id="45e56-123">Les identificateurs de propriété des propriétés nommées tombent dans la plage comprise entre 0x8000 et 0x8FFF.</span><span class="sxs-lookup"><span data-stu-id="45e56-123">The property identifiers of named properties fall in the range from 0x8000 to 0x8FFF.</span></span> 
  
<span data-ttu-id="45e56-124">Les propriétés nommées sont également attribuées par un jeu de propriétés, et soit un long ID (couvercle), qui est une valeur 32 bits non signée, ou une chaîne.</span><span class="sxs-lookup"><span data-stu-id="45e56-124">Named properties are additionally attributed by a property set, and either a long ID (LID), which is an unsigned 32-bit value, or a string.</span></span> <span data-ttu-id="45e56-125">Un jeu de propriétés est un GUID qui identifie un groupe de propriétés nommées avec un objectif similaire.</span><span class="sxs-lookup"><span data-stu-id="45e56-125">A property set is a GUID that identifies a group of named properties with a similar purpose.</span></span> <span data-ttu-id="45e56-126">Le jeu de propriétés et le couvercle ou le nom de la chaîne sont utilisés pour obtenir ou définir la propriété nommée.</span><span class="sxs-lookup"><span data-stu-id="45e56-126">The property set and LID or string name are used to get or set the named property.</span></span>

<span data-ttu-id="45e56-129">Pour les propriétés qui sont du type de chaîne, en fonction de la prise en charge d'Unicode dans la plateforme sous-jacente, la propriété peut être de type PT_STRING8 (chaîne de caractères 8 bits terminée par un caractère null) ou PT_UNICODE (chaîne Unicode terminée par null).</span><span class="sxs-lookup"><span data-stu-id="45e56-129">For properties that are of the string type, depending on the support for Unicode in the underlying platform, the property can be of type PT_STRING8 (null-terminated 8-bit character string) or PT_UNICODE (null-terminated Unicode string).</span></span> <span data-ttu-id="45e56-130">Une propriété peut être définie avec le type PT_TSTRING et en fonction des paramètres de compilation, PT_TSTRING par défaut une chaîne Unicode pour les plateformes qui prennent en charge Unicode ou une chaîne PT_STRING8 pour les plateformes qui prennent en charge ANSI ou DBCS.</span><span class="sxs-lookup"><span data-stu-id="45e56-130">A property can be defined with the PT_TSTRING type, and depending on compilation settings, PT_TSTRING defaults to a Unicode string for platforms that support Unicode, or to a PT_STRING8 string for platforms that support ANSI or DBCS.</span></span> <span data-ttu-id="45e56-131">Il est courant qu'une propriété de type chaîne soit identifiée par trois noms similaires, tels que **PR_ACCOUNT**, **PR_ACCOUNT_A**et **PR_ACCOUNT_W**, respectivement du type PT_TSTRING, PT_STRING8 et PT_UNICODE.</span><span class="sxs-lookup"><span data-stu-id="45e56-131">It is common that a string-typed property is identified by three similar names, such as **PR_ACCOUNT**, **PR_ACCOUNT_A**, and **PR_ACCOUNT_W**, which are of the type PT_TSTRING, PT_STRING8, and PT_UNICODE respectively.</span></span>
