# <a name="pidtagcontactaddressbookuid-canonical-property"></a>Propriété canonique PidTagContactAddressBookUid

  
  
**S’applique à**: Outlook 
  
Contient l’ID d’utilisateur qui identifie la session du carnet d’adresses de contacts.
  
|||
|:-----|:-----|
|Propriétés associées :  <br/> |PR_CONTAB_UID  <br/> |
|Identificateur :  <br/> |0x6601  <br/> |
|Type de données :  <br/> |PT_BINARY  <br/> |
|Zone :  <br/> |Carnet d’adresses de contacts  <br/> |
   
## <a name="remarks"></a>Remarques

Si vous avez plusieurs carnets d’adresses de contacts configuré, cette propriété doit être définie pour chacun d’eux.
