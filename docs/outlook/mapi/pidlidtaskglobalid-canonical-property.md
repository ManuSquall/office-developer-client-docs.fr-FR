# <a name="pidlidtaskglobalid-canonical-property"></a>Propriété canonique PidLidTaskGlobalId

  
  
**S’applique à** : Outlook 2013 | Outlook 2016 
  
Localise une tâche existante à l'aide d'un GUID, lors de la réception d'une réponse de tâche ou d'une mise à jour de tâche.
  
|||
|:-----|:-----|
|Propriétés associées :  <br/> |dispidTaskGlobalObjId  <br/> |
|Jeu de propriétés:  <br/> |PSETID_Common  <br/> |
|ID long (couvercle):  <br/> |0x00008519  <br/> |
|Type de données :  <br/> |PT_BINARY  <br/> |
|Domaine :  <br/> |Tâche  <br/> |
   
## <a name="remarks"></a>Remarques

Cette propriété n'est pas définie pour les tâches non affectées.
