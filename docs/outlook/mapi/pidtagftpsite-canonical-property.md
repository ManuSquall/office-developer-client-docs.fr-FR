# <a name="pidtagftpsite-canonical-property"></a>Propriété canonique PidTagFtpSite

  
  
**S’applique à** : Outlook 2013 | Outlook 2016 
  
Contient l’URL de protocole FTP (File Transfer) du contact. FTP est un protocole utilisé pour transférer des données, tel que spécifié dans [RFC959].
  
|||
|:-----|:-----|
|Propriétés associées :  <br/> |PR_FTP_SITE, PR_FTP_SITE_A, PR_FTP_SITE_W  <br/> |
|Identificateur :  <br/> |0x3A4C  <br/> |
|Type de données :  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Domaine :  <br/> |Utilisateur de messagerie MAPI  <br/> |
