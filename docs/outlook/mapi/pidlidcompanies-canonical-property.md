# <a name="pidlidcompanies-canonical-property"></a>Propriété canonique PidLidCompanies

  
  
**S’applique à**: Outlook 
  
Contient les noms des sociétés qui sont associées à un élément.
  
|||
|:-----|:-----|
|Propriétés associées :  <br/> |dispidCompanies  <br/> |
|Jeu de propriétés :  <br/> |PSETID_Common  <br/> |
|ID de type long (capot) :  <br/> |0x00008539  <br/> |
|Type de données :  <br/> |PT_MV_UNICODE  <br/> |
|Zone :  <br/> |Général de messagerie  <br/> |
