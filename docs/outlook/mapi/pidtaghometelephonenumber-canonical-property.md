# <a name="pidtaghometelephonenumber-canonical-property"></a>Propriété canonique PidTagHomeTelephoneNumber

  
  
**S’applique à**: Outlook 
  
Contient le numéro de téléphone principal d’accueil du destinataire. 
  
|||
|:-----|:-----|
|Propriétés associées :  <br/> |PR_HOME_TELEPHONE_NUMBER, PR_HOME_TELEPHONE_NUMBER_A, PR_HOME_TELEPHONE_NUMBER_W  <br/> |
|Identificateur :  <br/> |0x3A09  <br/> |
|Type de données :  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Zone :  <br/> |Utilisateur de messagerie MAPI  <br/> |
   
## <a name="remarks"></a>Remarques

Ces propriétés fournissent une identification et accès pour un destinataire. Ils sont définis par le destinataire et de leur organisation.
