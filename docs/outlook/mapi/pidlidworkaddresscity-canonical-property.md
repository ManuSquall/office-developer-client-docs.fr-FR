# <a name="pidlidworkaddresscity-canonical-property"></a>Propriété canonique PidLidWorkAddressCity

  
  
**S’applique à** : Outlook 2013 | Outlook 2016 
  
Spécifie la partie ville ou localité d’adresse du contact.
  
|||
|:-----|:-----|
|Propriétés associées :  <br/> |dispidWorkAddressCity  <br/> |
|Jeu de propriétés :  <br/> |PSETID_Address  <br/> |
|ID de type long (capot) :  <br/> |0x00008046  <br/> |
|Type de données :  <br/> |PT_UNICODE  <br/> |
|Domaine :  <br/> |Contact  <br/> |
