# <a name="pidtagweddinganniversary-canonical-property"></a>Propriété canonique PidTagWeddingAnniversary

  
  
**S’applique à** : Outlook 2013 | Outlook 2016 
  
Indique la date de l'anniversaire de mariage d'un utilisateur.
  
|||
|:-----|:-----|
|Propriétés associées :  <br/> |PR_WEDDING_ANNIVERSARY  <br/> |
|Identificateur :  <br/> |0x3A41  <br/> |
|Type de données :  <br/> |PT_SYSTIME  <br/> |
|Domaine :  <br/> |Utilisateur de messagerie MAPI  <br/> |
   
## <a name="remarks"></a>Remarques

Cette propriété est définie par l'utilisateur ou l'organisation de l'utilisateur.
