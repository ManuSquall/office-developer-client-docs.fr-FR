# <a name="pidlidnoteheight-canonical-property"></a>Propriété canonique PidLidNoteHeight

  
  
**S’applique à**: Outlook 
  
Spécifie la hauteur de la fenêtre de message visible en pixels. 
  
|||
|:-----|:-----|
|Propriétés associées :  <br/> |dispidNoteHeight  <br/> |
|Jeu de propriétés :  <br/> |PSETID_Note  <br/> |
|ID de type long (capot) :  <br/> |0x00008B03  <br/> |
|Type de données :  <br/> |PT_LONG  <br/> |
|Zone :  <br/> |Pense-bête  <br/> |
   
## <a name="remarks"></a>Remarques

Cette valeur doit être supérieure à zéro.
