# <a name="pidtagassociatedsharingprovider-canonical-property"></a>Propriété canonique PidTagAssociatedSharingProvider

  
  
**S’applique à**: Outlook 
  
Détermine si un fournisseur de magasin de dossiers personnels (PST) de fichier est configuré comme un fichier PST de Microsoft SharePoint 2010.
  
|||
|:-----|:-----|
|Propriétés associées :  <br/> |PR_ASSOCIATED_SHARING_PROVIDER  <br/> |
|Identificateur :  <br/> |0x0EA00048  <br/> |
|Type de données :  <br/> |PT_CLSID  <br/> |
|Zone :  <br/> |Tableau de stockage personnel (.pst) interne  <br/> |
   
## <a name="remarks"></a>Remarques

Cette propriété indique si un fichier PST est configuré pour Microsoft SharePoint 2010. Si la propriété n’est pas présente, le fichier PST n’a pas été configuré comme un fichier PST de Microsoft SharePoint 2010.
