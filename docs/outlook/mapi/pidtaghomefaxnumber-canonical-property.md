# <a name="pidtaghomefaxnumber-canonical-property"></a>Propriété canonique PidTagHomeFaxNumber

  
  
**S’applique à** : Outlook 2013 | Outlook 2016 
  
Contient le numéro de téléphone de l’ordinateur du destinataire télécopie (domicile). 
  
|||
|:-----|:-----|
|Propriétés associées :  <br/> |NUMÉRO_FAX_PERSONNEL_PR, PR_HOME_FAX_NUMBER_A, PR_HOME_FAX_NUMBER_W  <br/> |
|Identificateur :  <br/> |0x3A25  <br/> |
|Type de données :  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Domaine :  <br/> |Utilisateur de messagerie MAPI  <br/> |
   
## <a name="remarks"></a>Remarques

Ces propriétés permettent d’identifier et accéder aux informations sur un destinataire. Ils sont définis par le destinataire et de leur organisation.
