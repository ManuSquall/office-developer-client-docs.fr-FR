# <a name="pidtagoriginalsenderemailaddress-canonical-property"></a>Propriété canonique PidTagOriginalSenderEmailAddress

  
  
**S’applique à**: Outlook 2013 | Outlook 2016 
  
Contient l’adresse de messagerie de l’expéditeur de la première version d’un message, autrement dit, le message avant d’être transférés ou une réponse.
  
|||
|:-----|:-----|
|Propriétés associées :  <br/> |PR_ORIGINAL_SENDER_EMAIL_ADDRESS, PR_ORIGINAL_SENDER_EMAIL_ADDRESS_A, PR_ORIGINAL_SENDER_EMAIL_ADDRESS_W  <br/> |
|Identificateur :  <br/> |0x0067  <br/> |
|Type de données :  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Domaine :  <br/> |Général de messagerie  <br/> |
   
## <a name="remarks"></a>Remarques

Ces propriétés sont des exemples de propriétés d’adresse de l’expéditeur d’origine d’un message. Au premier envoi du message, l’application cliente doit définir cette propriété à la valeur de **PR_SENDER_EMAIL_ADDRESS** ([PidTagSenderEmailAddress](pidtagsenderemailaddress-canonical-property.md)). Il n’est jamais modifié lorsque le message est transféré ou d’une réponse.
