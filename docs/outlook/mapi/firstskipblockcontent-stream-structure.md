# <a name="firstskipblockcontent-stream-structure"></a>Structure de flux FirstSkipBlockContent

  
  
**S’applique à**: Outlook 2013 | Outlook 2016 
  
La structure de flux FirstSkipBlockContent est le contenu de la première structure [SkipBlock](skipblock-stream-structure.md) dans l’élément de données SkipBlocks d’un flux [FieldDefinition](fielddefinition-stream-structure.md) . Le flux FirstSkipBlockContent est simplement un seul élément de données, FieldName : 
  
- De FieldName : [PackedUnicodeString](packedunicodestring-stream-structure.md), le nom du champ.
