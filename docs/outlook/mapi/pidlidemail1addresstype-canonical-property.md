# <a name="pidlidemail1addresstype-canonical-property"></a>Propriété canonique PidLidEmail1AddressType

  
  
**S’applique à** : Outlook 2013 | Outlook 2016 
  
Spécifie le type d'adresse de la première adresse de messagerie.
  
|||
|:-----|:-----|
|Propriétés associées :  <br/> |dispidEmail1AddrType  <br/> |
|Jeu de propriétés:  <br/> |PSETID_Address  <br/> |
|ID long (couvercle):  <br/> |0x00008082  <br/> |
|Type de données :  <br/> |PT_UNICODE  <br/> |
|Domaine :  <br/> |Contact  <br/> |
   
## <a name="remarks"></a>Remarques

S'il est présent, la valeur de cette propriété doit être un type d'adresse valide.
