# <a name="cbnewflatentry"></a>CbNewFLATENTRY

  
  
**S’applique à**: Outlook 
  
Calcule le nombre d’octets qui doit être alloué pour une nouvelle structure [FLATENTRY](flatentry.md) qui contient un identificateur d’entrée d’une taille spécifiée en octets. 
  
|||
|:-----|:-----|
|Fichier d’en-tête :  <br/> |Mapidefs.h  <br/> |
|Structure connexe :  <br/> |**FLATENTRY** <br/> |
   
```cpp
CbNewFLATENTRY (_cb)
```

## <a name="parameters"></a>Paramètres

 __cb_
  
> Nombre d’octets de l’identificateur d’entrée à inclure dans la nouvelle structure **FLATENTRY** .
