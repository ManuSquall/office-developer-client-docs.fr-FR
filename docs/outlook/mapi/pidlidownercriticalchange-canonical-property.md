# <a name="pidlidownercriticalchange-canonical-property"></a>Propriété canonique PidLidOwnerCriticalChange

  
  
**S’applique à**: Outlook 
  
Spécifie la date et l’heure lorsqu’une demande de réunion a été envoyée par l’organisateur.
  
|||
|:-----|:-----|
|Propriétés associées :  <br/> |LID_OWNER_CRITICAL_CHANGE  <br/> |
|Jeu de propriétés :  <br/> |PSETID_Meeting  <br/> |
|ID de type long (capot) :  <br/> |0x0000001A  <br/> |
|Type de données :  <br/> |PT_SYSTIME  <br/> |
|Zone :  <br/> |Réunions  <br/> |
   
## <a name="remarks"></a>Remarques

La valeur doit être spécifiée dans le temps universel coordonné (UTC).
