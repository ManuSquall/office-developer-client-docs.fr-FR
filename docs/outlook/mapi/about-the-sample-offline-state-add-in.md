# <a name="about-the-sample-offline-state-add-in"></a>À propos de l’exemple en mode hors connexion-dans l’état

  
  
**S’applique à**: Outlook 
  
L’API d’état en mode hors connexion prend en charge les rappels indiquant les modifications de l’état de connexion d’un utilisateur dans Outlook, par exemple, d’en cours en ligne dans Outlook pour en cours en mode hors connexion. Le complément exemple hors connexion état est un complément COM écrit en langage C++ qui montre comment recevoir des notifications de changements d’état de connexion et comment modifier l’état actuel de l’API de l’état en mode hors connexion. Pour plus d’informations sur l’API de l’état en mode hors connexion, voir [à propos en mode hors connexion état API](about-the-offline-state-api.md).
