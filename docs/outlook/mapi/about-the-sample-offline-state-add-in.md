# <a name="about-the-sample-offline-state-add-in"></a>À propos de l'exemple de complément d'État hors connexion

  
  
**S’applique à** : Outlook 2013 | Outlook 2016 
  
L'API d'État hors connexion prend en charge les rappels indiquant les modifications apportées à l'état de connexion d'un utilisateur dans Outlook (par exemple, en ligne dans Outlook pour être en mode hors connexion). L'exemple de complément d'État hors connexion est un complément COM écrit en C++, qui montre comment recevoir des notifications de modification de l'état de la connexion et comment modifier l'état actuel à l'aide de l'API d'État hors connexion. Pour plus d’informations sur l’API d’état hors connexion, reportez-vous à l’article [À propos de l’API d’état hors connexion](about-the-offline-state-api.md).
