# <a name="pidtagscheduleinfodelegatorwantscopy-canonical-property"></a>Propriété canonique PidTagScheduleInfoDelegatorWantsCopy

  
  
**S’applique à** : Outlook 2013 | Outlook 2016 
  
Contient la valeur TRUE si la personne qui souhaite recevoir des copies des objets liés à la réunion est envoyée au délégué.
  
|||
|:-----|:-----|
|Propriétés associées :  <br/> |PR_SCHDINFO_BOSS_WANTS_COPY  <br/> |
|Identificateur :  <br/> |0x6842  <br/> |
|Type de données :  <br/> |PT_BOOLEAN  <br/> |
|Domaine :  <br/> |Disponibilité  <br/> |
   
## <a name="remarks"></a>Remarques

Cette propriété doit être définie dans l'objet information de délégué.
