# <a name="pidtagbusinessfaxnumber-canonical-property"></a>Propriété canonique PidTagBusinessFaxNumber

  
  
**S’applique à** : Outlook 2013 | Outlook 2016 
  
Contient le numéro de téléphone de l'ordinateur de télécopie professionnel du destinataire. 
  
|||
|:-----|:-----|
|Propriétés associées :  <br/> |PR_BUSINESS_FAX_NUMBER, PR_BUSINESS_FAX_NUMBER_A, PR_BUSINESS_FAX_NUMBER_W  <br/> |
|Identificateur :  <br/> |0x3A24  <br/> |
|Type de données :  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Domaine :  <br/> |Contact  <br/> |
   
## <a name="remarks"></a>Remarques

Ces propriétés sont des exemples de propriétés qui fournissent des informations d'identification et d'accès à un destinataire. Ces propriétés sont définies par le destinataire et l'organisation du destinataire.
