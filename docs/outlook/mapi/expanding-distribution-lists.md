# <a name="expanding-distribution-lists"></a>Extension des listes de distribution

  
  
**S’applique à**: Outlook 2013 | Outlook 2016 
  
 **Pour demander de MAPI pour développer une liste de distribution**
  
- Définissez sa propriété **TYPEADR_PR** ([PidTagAddressType](pidtagaddresstype-canonical-property.md)) à MAPIPDL.
    
    MAPI développe les adresses de ce type avant d’envoyer le message vers le fournisseur de transport.
