# <a name="pidlidreferenceentryid-canonical-property"></a>Propriété canonique PidLidReferenceEntryId

  
  
**S’applique à** : Outlook 2013 | Outlook 2016 
  
Spécifie l' [ID](entryid.md) de référence du contact. 
  
|||
|:-----|:-----|
|Propriétés associées :  <br/> |dispidReferenceEID  <br/> |
|Jeu de propriétés:  <br/> |PSETID_Common  <br/> |
|ID long (couvercle):  <br/> |0x000085BD  <br/> |
|Type de données :  <br/> |PT_BINARY  <br/> |
|Domaine :  <br/> |Contact  <br/> |
   
## <a name="remarks"></a>Remarques

S'il est présent, cette propriété doit être égale à la valeur **EntryID** du contact.
