# <a name="pidtagservices-canonical-property"></a>Propriété canonique PidTagServices

  
  
**S’applique à**: Outlook 
  
Contient une liste d’identificateurs des services de messagerie dans le profil actif.
  
|||
|:-----|:-----|
|Propriétés associées :  <br/> |PR_SERVICES  <br/> |
|Identificateur :  <br/> |0x3D0E  <br/> |
|Type de données :  <br/> |PT_BINARY  <br/> |
|Zone :  <br/> |Profil MAPI  <br/> |
   
## <a name="remarks"></a>Remarques

N’utilisez pas cette propriété. Il est réservé pour une utilisation par MAPI.
