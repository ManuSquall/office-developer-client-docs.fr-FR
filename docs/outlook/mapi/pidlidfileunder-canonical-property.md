# <a name="pidlidfileunder-canonical-property"></a>Propriété canonique PidLidFileUnder

  
  
**S’applique à**: Outlook 2013 | Outlook 2016 
  
Spécifie le nom sous lequel le contact est déposé lors de l’affichage d’une liste de contacts.
  
|||
|:-----|:-----|
|Propriétés associées :  <br/> |dispidFileUnder  <br/> |
|Jeu de propriétés :  <br/> |PSETID_Address  <br/> |
|ID de type long (capot) :  <br/> |0x00008005  <br/> |
|Type de données :  <br/> |PT_UNICODE  <br/> |
|Domaine :  <br/> |Contact  <br/> |
   
## <a name="remarks"></a>Remarques

L’application doit traiter cette propriété comme un PT_UNICODE vide s’il est absent du contact.
