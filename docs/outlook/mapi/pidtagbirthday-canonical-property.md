# <a name="pidtagbirthday-canonical-property"></a>Propriété canonique PidTagBirthday

  
  
**S’applique à** : Outlook 2013 | Outlook 2016 
  
Contient la date d’anniversaire du contact.
  
|||
|:-----|:-----|
|Propriétés associées :  <br/> |PR_BIRTHDAY  <br/> |
|Identificateur :  <br/> |0x3A42  <br/> |
|Type de données :  <br/> |PT_SYSTIME  <br/> |
|Domaine :  <br/> |Contact  <br/> |
   
## <a name="remarks"></a>Remarques

Cette propriété est une des propriétés qui fournit des informations d’identification et l’accès sur un contact. Ces propriétés sont définies par l’utilisateur et d’organisation de l’utilisateur.
