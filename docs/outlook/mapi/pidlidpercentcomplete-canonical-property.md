# <a name="pidlidpercentcomplete-canonical-property"></a>Propriété canonique PidLidPercentComplete

  
  
**S’applique à**: Outlook 2013 | Outlook 2016 
  
Indique la progression de l’utilisateur a effectué sur une tâche. 
  
|||
|:-----|:-----|
|Propriétés associées :  <br/> |dispidPercentComplete  <br/> |
|Jeu de propriétés :  <br/> |PSETID_Task  <br/> |
|ID de type long (capot) :  <br/> |0x00008102  <br/> |
|Type de données :  <br/> |PT_R8  <br/> |
|Domaine :  <br/> |Task  <br/> |
   
## <a name="remarks"></a>Remarques

La valeur de cette propriété doit être un nombre supérieur ou égal à 0,0 et inférieure ou égale à 1.0, où 1.0 indique que le travail est terminé et 0.0 indique que le travail n’a pas commencé.
  
Pour un objet de message avec indicateur de temps, la valeur de cette propriété doit être définie sur 1.0 si l’objet est marquée terminée et doit être définie à 0.0 dans le cas contraire.
