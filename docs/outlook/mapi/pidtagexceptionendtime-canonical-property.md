# <a name="pidtagexceptionendtime-canonical-property"></a>Propriété canonique PidTagExceptionEndTime

  
  
**S’applique à**: Outlook 2013 | Outlook 2016 
  
Indique la date de fin et l’heure de l’exception dans le fuseau horaire local de l’ordinateur lorsque l’exception est créée.
  
|||
|:-----|:-----|
|Propriétés associées :  <br/> |PR_EXCEPTION_ENDTIME  <br/> |
|Identificateur :  <br/> |0x7FFC  <br/> |
|Type de données :  <br/> |PT_SYSTIME  <br/> |
|Domaine :  <br/> |Défini par la classe de message non transmissible  <br/> |
   
## <a name="remarks"></a>Remarques

> [!NOTE]
> Cette propriété est une information et ne doit pas être utilisée pour des informations importantes.
