# <a name="pidlidoriginalstoreentryid-canonical-property"></a>Propriété canonique PidLidOriginalStoreEntryId

  
  
**S’applique à**: Outlook 2013 | Outlook 2016 
  
Spécifie l’identificateur d’entrée du magasin de la personne.
  
|||
|:-----|:-----|
|Propriétés associées :  <br/> |dispidOrigStoreEid  <br/> |
|Jeu de propriétés :  <br/> |PSETID_Appointment  <br/> |
|ID de type long (capot) :  <br/> |0x00008237  <br/> |
|Type de données :  <br/> |PT_BINARY  <br/> |
|Domaine :  <br/> |Réunions  <br/> |
   
## <a name="remarks"></a>Remarques

Cette propriété doit être définie sur les objets de la conférence qui ont été créées ou mises à jour par un délégué.
