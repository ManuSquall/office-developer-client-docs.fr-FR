# <a name="pidlidfax3emailaddress-canonical-property"></a>Propriété canonique PidLidFax3EmailAddress

  
  
**S’applique à**: Outlook 
  
Indique l’adresse de messagerie du contact du autre adresse télécopie.
  
|||
|:-----|:-----|
|Propriétés associées :  <br/> |dispidFax3EmailAddress  <br/> |
|Jeu de propriétés :  <br/> |PSETID_Address  <br/> |
|ID de type long (capot) :  <br/> |0x000080D3  <br/> |
|Type de données :  <br/> |PT_UNICODE  <br/> |
|Zone :  <br/> |Contact  <br/> |
   
## <a name="remarks"></a>Remarques

Cette propriété, le cas échéant, doit contenir un nom complet explicite de l’utilisateur, suivi par le « @ » caractère, suivi d’un numéro de télécopie.
