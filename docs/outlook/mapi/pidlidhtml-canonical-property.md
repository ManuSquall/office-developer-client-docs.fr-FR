# <a name="pidlidhtml-canonical-property"></a>Propriété canonique PidLidHtml

  
  
**S’applique à** : Outlook 2013 | Outlook 2016 
  
Spécifie l’URL de page Web d’entreprise du contact.
  
|||
|:-----|:-----|
|Propriétés associées :  <br/> |dispidHTML  <br/> |
|Jeu de propriétés :  <br/> |PSETID_Address  <br/> |
|ID de type long (capot) :  <br/> |0x0000802B  <br/> |
|Type de données :  <br/> |PT_UNICODE  <br/> |
|Domaine :  <br/> |Contact  <br/> |
   
## <a name="remarks"></a>Remarques

La valeur de cette propriété, le cas échéant, doit être identique à la valeur de la propriété **PR_BUSINESS_HOME_PAGE** ([PidTagBusinessHomePage](pidtagbusinesshomepage-canonical-property.md)).
