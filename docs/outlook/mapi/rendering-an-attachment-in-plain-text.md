<span data-ttu-id="78c7e-105">Pour afficher une pièce jointe dans un message en texte brut, récupérez la propriété **PR_RENDERING_POSITION** ([PidTagRenderingPosition](pidtagrenderingposition-canonical-property.md)) de la pièce jointe et appliquez-la aux données de **PR_ATTACH_RENDERING** ([PidTagAttachRendering](pidtagattachrendering-canonical-property.md)) inspecteur.</span><span class="sxs-lookup"><span data-stu-id="78c7e-105">To render an attachment in a message with plain text, retrieve the attachment's **PR_RENDERING_POSITION** ([PidTagRenderingPosition](pidtagrenderingposition-canonical-property.md)) property and apply it to the data in the **PR_ATTACH_RENDERING** ([PidTagAttachRendering](pidtagattachrendering-canonical-property.md)) property.</span></span> <span data-ttu-id="78c7e-106">Il existe deux façons de récupérer **PR_RENDERING_POSITION**:</span><span class="sxs-lookup"><span data-stu-id="78c7e-106">There are two ways to retrieve **PR_RENDERING_POSITION**:</span></span>
