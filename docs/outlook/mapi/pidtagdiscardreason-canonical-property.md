# <a name="pidtagdiscardreason-canonical-property"></a>Propriété canonique PidTagDiscardReason

  
  
**S’applique à**: Outlook 2013 | Outlook 2016 
  
Contient un motif pourquoi un message transfert agent d’a supprimé un message. 
  
|||
|:-----|:-----|
|Propriétés associées :  <br/> |PR_DISCARD_REASON  <br/> |
|Identificateur :  <br/> |0x0011  <br/> |
|Type de données :  <br/> |PT_LONG  <br/> |
|Domaine :  <br/> |Enveloppe MAPI  <br/> |
   
## <a name="remarks"></a>Remarques

La raison contenue dans cette propriété est utilisée dans un rapport de non-remise pour le message.
