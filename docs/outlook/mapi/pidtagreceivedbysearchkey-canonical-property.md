# <a name="pidtagreceivedbysearchkey-canonical-property"></a>Propriété canonique PidTagReceivedBySearchKey

  
  
**S’applique à**: Outlook 2013 | Outlook 2016 
  
Contient la clé de recherche de l’utilisateur de messagerie qui reçoit le message.
  
|||
|:-----|:-----|
|Propriétés associées :  <br/> |PR_RECEIVED_BY_SEARCH_KEY  <br/> |
|Identificateur :  <br/> |0x0051  <br/> |
|Type de données :  <br/> |PT_BINARY  <br/> |
|Domaine :  <br/> |Address  <br/> |
   
## <a name="remarks"></a>Remarques

Cette propriété est une des propriétés d’adresse de l’utilisateur de messagerie qui reçoit le message. Elle doit être définie par le fournisseur de transport entrant.
