# <a name="pidnamewordcount-canonical-property"></a>Propriété canonique PidNameWordCount

  
  
**S’applique à** : Outlook 2013 | Outlook 2016 
  
Spécifie le nombre de mots du fichier qui est attaché au document.
  
|||
|:-----|:-----|
|Noms conviviaux :  <br/> |Aucune  <br/> |
|Jeu de propriétés :  <br/> |PS_PUBLIC_STRINGS  <br/> |
|Nom de la propriété :  <br/> |WordCount  <br/> |
|Type de données :  <br/> |PT_LONG  <br/> |
|Domaine :  <br/> |Common  <br/> |
