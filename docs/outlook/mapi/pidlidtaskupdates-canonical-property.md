# <a name="pidlidtaskupdates-canonical-property"></a>Propriété canonique PidLidTaskUpdates

  
  
**S’applique à**: Outlook 2013 | Outlook 2016 
  
Indique si le destinataire de la tâche a été demandé pour envoyer une mise à jour de tâches lorsque la tâche est modifiée.
  
|||
|:-----|:-----|
|Propriétés associées :  <br/> |dispidTaskUpdates  <br/> |
|Jeu de propriétés :  <br/> |PSETID_Task  <br/> |
|ID de type long (capot) :  <br/> |0x0000811B  <br/> |
|Type de données :  <br/> |PT_BOOLEAN  <br/> |
|Domaine :  <br/> |Task  <br/> |
