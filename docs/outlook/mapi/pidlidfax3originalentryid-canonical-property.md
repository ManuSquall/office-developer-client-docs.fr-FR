# <a name="pidlidfax3originalentryid-canonical-property"></a>Propriété canonique PidLidFax3OriginalEntryId

  
  
**S’applique à**: Outlook 
  
Indique la propriété EntryID d’origine du contact du autre adresse télécopie.
  
|||
|:-----|:-----|
|Propriétés associées :  <br/> |dispidFax3OriginalEntryID  <br/> |
|Jeu de propriétés :  <br/> |PSETID_Address  <br/> |
|ID de type long (capot) :  <br/> |0x000080D5  <br/> |
|Type de données :  <br/> |PT_BINARY  <br/> |
|Zone :  <br/> |Contact  <br/> |
   
## <a name="remarks"></a>Remarques

Cette propriété, le cas échéant, doit spécifier EntryId unique qui correspond à cette adresse télécopie.
