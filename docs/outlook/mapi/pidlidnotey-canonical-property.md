# <a name="pidlidnotey-canonical-property"></a>Propriété canonique PidLidNoteY

  
  
**S’applique à**: Outlook 
  
Indique la distance, en pixels, entre le bord supérieur de l’écran où une interface utilisateur affiche une note.
  
|||
|:-----|:-----|
|Propriétés associées :  <br/> |dispidNoteY  <br/> |
|Jeu de propriétés :  <br/> |PSETID_Note  <br/> |
|ID de type long (capot) :  <br/> |0x00008B05  <br/> |
|Type de données :  <br/> |PT_LONG  <br/> |
|Zone :  <br/> |Pense-bête  <br/> |
