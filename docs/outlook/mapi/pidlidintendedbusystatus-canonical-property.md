# <a name="pidlidintendedbusystatus-canonical-property"></a>Propriété canonique PidLidIntendedBusyStatus

  
  
**S’applique à** : Outlook 2013 | Outlook 2016 
  
Spécifie la valeur de la propriété **dispidBusyStatus** ([PidLidBusyStatus](pidlidbusystatus-canonical-property.md)) sur la réunion dans le calendrier de l’organisateur lorsque la demande de réunion ou une mise à jour de la réunion a été envoyé.
  
|||
|:-----|:-----|
|Propriétés associées :  <br/> |dispidIntendedBusyStatus  <br/> |
|Jeu de propriétés :  <br/> |PSETID_Appointment  <br/> |
|ID de type long (capot) :  <br/> |0x00008224  <br/> |
|Type de données :  <br/> |PT_LONG  <br/> |
|Domaine :  <br/> |Réunions  <br/> |
   
## <a name="remarks"></a>Remarques

Les valeurs possibles de cette propriété sont les mêmes que ceux de la propriété **dispidBusyStatus**.
