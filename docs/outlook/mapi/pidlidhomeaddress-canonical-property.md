# <a name="pidlidhomeaddress-canonical-property"></a>Propriété canonique PidLidHomeAddress

  
  
**S’applique à**: Outlook 
  
Spécifie l’adresse complète du domicile du contact.
  
|||
|:-----|:-----|
|Propriétés associées :  <br/> |dispidHomeAddress  <br/> |
|Jeu de propriétés :  <br/> |PSETID_Address  <br/> |
|ID de type long (capot) :  <br/> |0x0000801A  <br/> |
|Type de données :  <br/> |PT_UNICODE  <br/> |
|Zone :  <br/> |Contact  <br/> |
   
## <a name="remarks"></a>Note

Cette propriété doit être une combinaison des autres propriétés de l’adresse physique et est basée sur les paramètres régionaux du client.
