# <a name="pidlidtimezonedescription-canonical-property"></a>Propriété canonique PidLidTimeZoneDescription

  
  
**S’applique à**: Outlook 
  
Spécifie une description de la chaîne du fuseau horaire.
  
|||
|:-----|:-----|
|Propriétés associées :  <br/> |dispidTimeZoneDesc  <br/> |
|Jeu de propriétés :  <br/> |PSETID_Appointment  <br/> |
|ID de type long (capot) :  <br/> |0x00008234  <br/> |
|Type de données :  <br/> |PT_UNICODE  <br/> |
|Zone :  <br/> |Calendrier  <br/> |
   
## <a name="remarks"></a>Remarques

Cette propriété spécifie une description explicite du fuseau horaire qui est représenté par les données dans la propriété **dispidTimeZoneStruct** ([PidLidTimeZoneStruct](pidlidtimezonestruct-canonical-property.md)).
