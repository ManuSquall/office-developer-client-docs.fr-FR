# <a name="pidlidtaskrecurrence-canonical-property"></a>Propriété canonique PidLidTaskRecurrence

  
  
**S’applique à**: Outlook 2013 | Outlook 2016 
  
Contient une structure RecurrencePattern qui fournit des informations sur les tâches répétitives.
  
|||
|:-----|:-----|
|Propriétés associées :  <br/> |dispidTaskRecur  <br/> |
|Jeu de propriétés :  <br/> |PSETID_Task  <br/> |
|ID de type long (capot) :  <br/> |0x00008116  <br/> |
|Type de données :  <br/> |PT_BINARY  <br/> |
|Domaine :  <br/> |Task  <br/> |
   
## <a name="remarks"></a>Remarques

Pour plus d’informations sur la façon de créer et de spécifier une structure RecurrencePattern, voir [créer un élément de tâche périodique Simple](how-to-create-a-simple-recurrent-task-item.md).
