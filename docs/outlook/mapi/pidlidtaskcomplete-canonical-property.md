# <a name="pidlidtaskcomplete-canonical-property"></a>Propriété canonique PidLidTaskComplete

  
  
**S’applique à** : Outlook 2013 | Outlook 2016 
  
Indique que la tâche est terminée.
  
|||
|:-----|:-----|
|Propriétés associées :  <br/> |dispidTaskComplete  <br/> |
|Jeu de propriétés :  <br/> |PSETID_Task  <br/> |
|ID de type long (capot) :  <br/> |0x0000811C  <br/> |
|Type de données :  <br/> |PT_BOOLEAN  <br/> |
|Domaine :  <br/> |Task  <br/> |
