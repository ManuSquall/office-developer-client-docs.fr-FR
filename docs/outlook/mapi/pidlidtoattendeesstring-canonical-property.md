# <a name="pidlidtoattendeesstring-canonical-property"></a>Propriété canonique PidLidToAttendeesString

  
  
**S’applique à**: Outlook 2013 | Outlook 2016 
  
Contient une liste de tous les participants à envoyer qui sont également des participants obligatoires.
  
|||
|:-----|:-----|
|Propriétés associées :  <br/> |dispidToAttendeesString  <br/> |
|Jeu de propriétés :  <br/> |PSETID_Appointment  <br/> |
|ID de type long (capot) :  <br/> |0x0000823B  <br/> |
|Type de données :  <br/> |PT_UNICODE  <br/> |
|Domaine :  <br/> |Réunions  <br/> |
   
## <a name="remarks"></a>Remarques

La valeur de chaque participant est la propriété **PR_DISPLAY_NAME** ([PidTagDisplayName](pidtagdisplayname-canonical-property.md)) de carnet d’adresses du participant. Des entrées doivent être séparées par un point-virgule suivi d’un espace. Cette propriété n’est pas requise.
