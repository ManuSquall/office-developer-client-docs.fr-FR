# <a name="pidtagabdefaultdir-canonical-property"></a>Propriété canonique PidTagAbDefaultDir

  
  
**S’applique à**: Outlook 
  
Contient l’identificateur d’entrée du conteneur de carnet d’adresses pour afficher le premier. 
  
|||
|:-----|:-----|
|Propriétés associées :  <br/> |PR_AB_DEFAULT_DIR  <br/> |
|Identificateur :  <br/> |0x3D06  <br/> |
|Type de données :  <br/> |PT_BINARY  <br/> |
|Zone :  <br/> |Carnet d’adresses  <br/> |
   
## <a name="remarks"></a>Remarques

N’utilisez pas cette propriété. Il est réservé pour une utilisation par MAPI.
