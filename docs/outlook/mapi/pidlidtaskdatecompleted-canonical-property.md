# <a name="pidlidtaskdatecompleted-canonical-property"></a>Propriété canonique PidLidTaskDateCompleted

  
  
**S’applique à**: Outlook 
  
Spécifie la date lorsque l’utilisateur termine la tâche.
  
|||
|:-----|:-----|
|Propriétés associées :  <br/> |dispidTaskDateCompleted  <br/> |
|Jeu de propriétés :  <br/> |PSETID_Task  <br/> |
|ID de type long (capot) :  <br/> |0x0000810F  <br/> |
|Type de données :  <br/> |PT_SYSTIME  <br/> |
|Zone :  <br/> |Tâche  <br/> |
   
## <a name="remarks"></a>Remarques

Si la valeur, cette propriété doit avoir un composant de minuit dans le fuseau horaire local, converti en temps universel coordonné (UTC).
