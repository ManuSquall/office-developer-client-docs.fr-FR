# <a name="pidtaghomeaddresscity-canonical-property"></a>Propriété canonique PidTagHomeAddressCity

  
  
**S’applique à** : Outlook 2013 | Outlook 2016 
  
Contient la ville de l'adresse personnelle du destinataire.
  
|||
|:-----|:-----|
|Propriétés associées :  <br/> |PR_HOME_ADDRESS_CITY, PR_HOME_ADDRESS_CITY_A, PR_HOME_ADDRESS_CITY_W  <br/> |
|Identificateur :  <br/> |0x3A59  <br/> |
|Type de données :  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Domaine :  <br/> |Utilisateur de messagerie MAPI  <br/> |
