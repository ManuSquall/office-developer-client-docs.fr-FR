# <a name="pidlidchangehighlight-canonical-property"></a>Propriété canonique PidLidChangeHighlight

  
  
**S’applique à**: Outlook 2013 | Outlook 2016 
  
Spécifie un champ de bits qui indique la modification de l’objet de la réunion.
  
|||
|:-----|:-----|
|Propriétés associées :  <br/> |dispidChangeHighlight  <br/> |
|Jeu de propriétés :  <br/> |PSETID_Appointment  <br/> |
|ID de type long (capot) :  <br/> |0x00008204  <br/> |
|Type de données :  <br/> |PT_LONG  <br/> |
|Domaine :  <br/> |Réunions  <br/> |
   
## <a name="remarks"></a>Remarques

Cette propriété n’est pas requise. Les indicateurs individuels qui peuvent être définies sont détaillées dans [[MS-OXOCAL]](http://msdn.microsoft.com/library/09861fde-c8e4-4028-9346-e7c214cfdba1%28Office.15%29.aspx).
