# <a name="pidlidemail2displayname-canonical-property"></a>Propriété canonique PidLidEmail2DisplayName

  
  
**S’applique à**: Outlook 
  
Spécifie le nom d’affichage lisible par l’utilisateur pour la deuxième adresse de messagerie.
  
|||
|:-----|:-----|
|Propriétés associées :  <br/> |dispidEmail2DisplayName  <br/> |
|Jeu de propriétés :  <br/> |PSETID_Address  <br/> |
|ID de type long (capot) :  <br/> |0x00008090  <br/> |
|Type de données :  <br/> |PT_UNICODE  <br/> |
|Zone :  <br/> |Contact  <br/> |
