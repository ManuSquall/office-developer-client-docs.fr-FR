# <a name="pidlidfax2displayname-canonical-property"></a>Propriété canonique PidLidFax2DisplayName

  
  
**S’applique à** : Outlook 2013 | Outlook 2016 
  
Spécifie le nom complet de l'adresse de télécopie personnelle du contact.
  
|||
|:-----|:-----|
|Propriétés associées :  <br/> |dispidFax2DisplayName  <br/> |
|Jeu de propriétés:  <br/> |PSETID_Address  <br/> |
|ID long (couvercle):  <br/> |0x000080C0  <br/> |
|Type de données :  <br/> |PT_UNICODE  <br/> |
|Domaine :  <br/> |Contact  <br/> |
