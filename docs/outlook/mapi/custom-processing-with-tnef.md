# <a name="custom-processing-with-tnef"></a>Personnalisé de traitement avec TNEF

**S’applique à**: Outlook 2013 | Outlook 2016 
  
Fournisseurs de transport peuvent utiliser un traitement personnalisé pour traiter les propriétés d’une pièce jointe lui-même, transmettre séparément des pièces jointes ou les transmettre par le biais de modèle de pièce jointe du système de messagerie. TNEF utilise un mécanisme qui permet au fournisseur de transport envoyer les pièces jointes par rapport à du message et les reconnecter côté récepteur.
