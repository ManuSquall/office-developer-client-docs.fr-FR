# <a name="about-the-attachment-security-api"></a>À propos de la sécurité des pièces jointes API

  
  
**S’applique à**: Outlook 
  
L’API de sécurité des pièces jointes permet de détecter si une pièce jointe est considérée comme non sécurisée par Outlook et est bloquée pour l’affichage et l’indexation de solutions Outlook. Il fournit l’interface et les définitions suivantes.
  
Définitions
  
- [Constantes MAPI](mapi-constants.md)
    
Interface
  
- **[IAttachmentSecurity](iattachmentsecurityiunknown.md)**
