# <a name="about-the-attachment-security-api"></a>À propos de l’API de sécurité des pièces jointes

  
  
**S’applique à** : Outlook 2013 | Outlook 2016 
  
L'API de sécurité des pièces jointes permet aux solutions Outlook de déterminer si une pièce jointe est considérée comme non sécurisée par Outlook et est bloquée pour l'affichage et l'indexation. Il fournit les définitions et l'interface suivantes.
  
Définitions
  
- [Constantes MAPI](mapi-constants.md)
    
Interface
  
- **[IAttachmentSecurity](iattachmentsecurityiunknown.md)**
