# <a name="pidlidappointmentlastsequence-canonical-property"></a>Propriété canonique PidLidAppointmentLastSequence

  
  
**S’applique à** : Outlook 2013 | Outlook 2016 
  
Indique le dernier numéro de séquence qui a été envoyé à un participant dans la bibliothèque multimédia.
  
|||
|:-----|:-----|
|Propriétés associées :  <br/> |dispidApptLastSequence  <br/> |
|Jeu de propriétés :  <br/> |PSETID_Appointment  <br/> |
|ID de type long (capot) :  <br/> |0x00008203  <br/> |
|Type de données :  <br/> |PT_LONG  <br/> |
|Domaine :  <br/> |Réunions  <br/> |
   
## <a name="remarks"></a>Remarques

Cette propriété n’a aucune signification d’un participant.
