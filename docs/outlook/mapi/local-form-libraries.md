# <a name="local-form-libraries"></a>Bibliothèques de formulaires locaux

  
  
**S’applique à** : Outlook 2013 | Outlook 2016 
  
Les bibliothèques de formulaires locales sont stockées directement sur l'ordinateur d'un utilisateur, généralement dans un fichier nommé FRMCACHE. DAT. 
  
Les applications clientes peuvent accéder aux formulaires dans la bibliothèque de formulaires locale sans accéder aux ressources réseau. Les bibliothèques de formulaires locales constituent une exception à la règle selon laquelle les formulaires sont stockés dans des tables de contenu associées, car la bibliothèque de formulaires locale est simplement un fichier sur le disque qui ne fait partie d'aucune hiérarchie de dossiers MAPI.
