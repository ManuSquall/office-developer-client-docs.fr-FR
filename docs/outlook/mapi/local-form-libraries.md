# <a name="local-form-libraries"></a>Bibliothèques de formulaires locaux

  
  
**S’applique à**: Outlook 2013 | Outlook 2016 
  
Bibliothèques de formulaires local sont stockées directement sur l’ordinateur d’un utilisateur, généralement dans un fichier appelé FRMCACHE.DAT. 
  
Applications clientes peuvent accéder à des formulaires dans la bibliothèque de formulaires local sans accéder aux ressources réseau. Bibliothèques de formulaires local sont une exception à la règle que les formulaires sont stockés dans les tables de contenu associé, étant donné que la bibliothèque de formulaires local est simplement un fichier sur le disque qui ne fait pas partie d’une hiérarchie de dossier MAPI.
