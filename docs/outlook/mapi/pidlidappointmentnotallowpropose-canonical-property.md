# <a name="pidlidappointmentnotallowpropose-canonical-property"></a>Propriété canonique PidLidAppointmentNotAllowPropose

  
  
**S’applique à**: Outlook 
  
Indique si les participants ne sont pas autorisés à proposer une nouvelle date/heure de la réunion.
  
|||
|:-----|:-----|
|Propriétés associées :  <br/> |dispidApptNotAllowPropose  <br/> |
|Jeu de propriétés :  <br/> |PSETID_Appointment  <br/> |
|ID de type long (capot) :  <br/> |0x0000825A  <br/> |
|Type de données :  <br/> |PT_BOOLEAN  <br/> |
|Zone :  <br/> |Réunions  <br/> |
   
## <a name="remarks"></a>Remarques

La valeur FALSE, ou l’absence de cette propriété indique que les participants sont autorisés à proposer une nouvelle date/heure.
