# <a name="pidtaggeneration-canonical-property"></a>Propriété canonique PidTagGeneration

  
  
**S’applique à** : Outlook 2013 | Outlook 2016 
  
Contient une abréviation générationnel qui suit le nom complet du destinataire. 
  
|||
|:-----|:-----|
|Propriétés associées :  <br/> |PR_GENERATION, PR_GENERATION_A, PR_GENERATION_W  <br/> |
|Identificateur :  <br/> |0x3A05  <br/> |
|Type de données :  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Domaine :  <br/> |Utilisateur de messagerie MAPI  <br/> |
   
## <a name="remarks"></a>Remarques

Ces propriétés fournissent des informations d'identification et d'accès à un destinataire. Elles sont définies par le destinataire et leur organisation. 
  
Les valeurs communes sont les suivants: Jr., SR. et III.
