# <a name="pidlidclassificationkeep-canonical-property"></a>Propriété canonique PidLidClassificationKeep

  
  
**S’applique à**: Outlook 2013 | Outlook 2016 
  
Indique si la classification de message doit persister avec le message si le message est ou d’une réponse.
  
|||
|:-----|:-----|
|Propriétés associées :  <br/> |dispidClassKeep  <br/> |
|Jeu de propriétés :  <br/> |PSETID_Common  <br/> |
|ID de type long (capot) :  <br/> |0x000085BA  <br/> |
|Type de données :  <br/> |PT_BOOLEAN  <br/> |
|Domaine :  <br/> |Général de messagerie  <br/> |
   
## <a name="remarks"></a>Remarques

Pour plus d’informations, voir [comment déployer la Classification de Message pour Outlook 2007](http://msdn.microsoft.com/library/5a220424-edd5-4a21-b7fd-8106c23c3b39.aspx).
