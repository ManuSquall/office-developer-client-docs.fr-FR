# <a name="pidtagsubfolders-canonical-property"></a>Propriété canonique PidTagSubfolders

  
  
**S’applique à**: Outlook 
  
Contient la valeur TRUE si un dossier contient des sous-dossiers.
  
|||
|:-----|:-----|
|Propriétés associées :  <br/> |PR_SUBFOLDERS  <br/> |
|Identificateur :  <br/> |0x360A  <br/> |
|Type de données :  <br/> |PT_BOOLEAN  <br/> |
|Zone :  <br/> |Conteneur MAPI  <br/> |
   
## <a name="remarks"></a>Remarques

Banques de messages doivent fournir cette propriété pour tous les dossiers.
