# <a name="pidnamenotecount-canonical-property"></a>Propriété canonique PidNameNoteCount

  
  
**S’applique à**: Outlook 2013 | Outlook 2016 
  
Spécifie le nombre de notes dans le fichier qui est attaché au document.
  
|||
|:-----|:-----|
|Noms conviviaux :  <br/> |Aucun  <br/> |
|Jeu de propriétés :  <br/> |PS_PUBLIC_STRINGS  <br/> |
|Nom de la propriété :  <br/> |NoteCount  <br/> |
|Type de données :  <br/> |PT_LONG  <br/> |
|Domaine :  <br/> |Common  <br/> |
