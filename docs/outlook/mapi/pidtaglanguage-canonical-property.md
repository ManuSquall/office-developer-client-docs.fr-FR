# <a name="pidtaglanguage-canonical-property"></a>Propriété canonique PidTagLanguage

  
  
**S’applique à** : Outlook 2013 | Outlook 2016 
  
Contient une valeur qui indique la langue dans laquelle l'utilisateur de messagerie écrit les messages.
  
|||
|:-----|:-----|
|Propriétés associées :  <br/> |PR_LANGUAGE, PR_LANGUAGE_A, PR_LANGUAGE_W  <br/> |
|Identificateur :  <br/> |0x3A0C  <br/> |
|Type de données :  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Domaine :  <br/> |Address  <br/> |
   
## <a name="remarks"></a>Remarques

La chaîne contient un seul code de pays/région à deux caractères.
