# <a name="propsmtpssl"></a>PROP_SMTP_SSL

Indique s'il faut utiliser le protocole SSL (Secure Socket Layer) pour le compte SMTP.
  
## <a name="quick-info"></a>Informations rapides

|||
|:-----|:-----|
|Identificateur :  <br/> |0x0202  <br/> |
|Type de propriété:  <br/> |PT_DWORD  <br/> |
|Balise de propriété:  <br/> |0x02020003  <br/> |
|Access  <br/> |Lecture seule  <br/> |
   
## <a name="remarks"></a>Remarques

Si la valeur est zéro, le chiffrement SSL n'est pas utilisé, sinon, le chiffrement SSL.
