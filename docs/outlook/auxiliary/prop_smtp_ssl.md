# <a name="propsmtpssl"></a>PROP_SMTP_SSL

Spécifie s’il faut utiliser le protocole Secure Sockets Layer (SSL) pour le compte SMTP.
  
## <a name="quick-info"></a>Informations rapides

|||
|:-----|:-----|
|Identificateur :  <br/> |0 x 0202  <br/> |
|Type de propriété :  <br/> |PT_DWORD  <br/> |
|Balise de propriété :  <br/> |0x02020003  <br/> |
|Access :  <br/> |Lecture seule  <br/> |
   
## <a name="remarks"></a>Remarques

Un moyen de valeur zéro ne pas utiliser le chiffrement SSL, sinon, utiliser le chiffrement SSL.
