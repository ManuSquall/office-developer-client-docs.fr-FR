# <a name="about-the-data-degradation-layer-api"></a>À propos de la couche de dégradation de données API

L’API de couche données dégradation offre la possibilité de masquer le format natif sous-jacent d’un objet Outlook, permettre à un client pour accéder à l’objet dans un format de caractères préféré plutôt que dans le format natif de l’objet. Il fournit les définitions de constantes illustrées dans [l’une des constantes (couche de dégradation de données API)](constants-data-degradation-layer-api.md)et la fonction [HrCreateNewWrappedObject](hrcreatenewwrappedobject.md) qui renvoie un objet Outlook encapsulé dans un format spécifié.
