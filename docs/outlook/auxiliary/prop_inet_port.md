# <a name="propinetport"></a>PROP_INET_PORT

Représente le numéro de port pour une boîte aux lettres Internet général.
  
## <a name="quick-info"></a>Informations rapides

|||
|:-----|:-----|
|Identificateur :  <br/> |0 x 0104  <br/> |
|Type de propriété :  <br/> |PT_DWORD  <br/> |
|Balise de propriété :  <br/> |0x01040003  <br/> |
|Access :  <br/> |Lecture seule  <br/> |
   
## <a name="remarks"></a>Remarques

110 pour des comptes POP est un exemple.
