|**Type de données**|**Description**|
|:-----|:-----|
|[ACCT_BIN](acct_bin.md) <br/> |Une variable de ce type de données contient une valeur binaire.  <br/> |
|[ACCT_VARIANT](acct_variant.md) <br/> |Une variable de ce type de données contienne la valeur de propriété, qui est un type de données variant.  <br/> |
