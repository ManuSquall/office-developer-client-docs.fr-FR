# <a name="data-types-freebusy-api"></a>Types de données (API de disponibilité)

Cette section décrit les types de données dans l'API de disponibilité.
  
|**Type de données**|**Description**|
|:-----|:-----|
|[FBBlock_1](fbblock_1.md) <br/> |Définit un bloc de données de disponibilité.  <br/> |
|[FBStatus](fbstatus.md) <br/> |Une énumération pour l'état de disponibilité des blocs de disponibilité.  <br/> |
|[FBUser](fbuser.md) <br/> |Identifie un utilisateur dont les données de disponibilité sont disponibles ou non.  <br/> |
